# 스토리북 씬 템플릿 (Storybook)

> **용도**: 소원이를 위한 맞춤 스토리북 페이지 생성
> **권장 모델**: Nano Banana (단일 캐릭터) / Pro (2명 이상)

---

## 프롬프트 템플릿

### 한글 설명
```
소원이의 기적 여정을 담은 동화책 페이지.
따뜻하고 희망적인 분위기, 수채화 동화책 느낌.
```

### 영문 프롬프트 (Gemini 입력용)

```
[STYLE LOCK]
2D hand-drawn children's book illustration, Ghibli-inspired warmth.
Soft watercolor on textured paper, dreamy color palette.
Gentle magical atmosphere, cozy and hopeful mood.
Hand-painted look, visible brush strokes, soft edges.

[SCENE]
A storybook page depicting {SCENE_DESCRIPTION}.
{CHARACTER} in {LOCATION} with {ACTION}.
Magical sparkles and soft light effects.
Space for text at {TEXT_POSITION}.

[COMPOSITION]
- Main character: {CHARACTER_POSITION}
- Background: {BACKGROUND_DESCRIPTION}
- Focal point: {FOCAL_POINT}
- Text area: {TOP/BOTTOM} {LEFT/RIGHT}

[MOOD]
Warm, hopeful, magical, encouraging.
Child-friendly but sophisticated illustration.

[NEGATIVE]
3D, photorealistic, scary elements, dark mood, detailed realistic face, text, logo, watermark.
```

## 페이지 타입별 템플릿

### 1. 오프닝 (시작 페이지)
```
A magical door opening to reveal a digital dragon palace.
Soft light streaming through, welcoming atmosphere.
{SOWNEE_NAME}의 기적 여정이 시작됩니다.
```

### 2. 소원 장면 (소원이 등장)
```
{CHARACTER} listening warmly to a wish.
Gentle expression, supportive posture.
Floating wish symbols around.
```

### 3. 응원 장면 (팀 지원)
```
Aurora5 team members sending encouragement.
Colorful energy flowing toward the center.
Stars and hearts as visual metaphors.
```

### 4. 클로징 (마무리)
```
{SOWNEE_NAME} surrounded by achieved dreams.
Celebration atmosphere, confetti and sparkles.
Happy ending mood.
```
