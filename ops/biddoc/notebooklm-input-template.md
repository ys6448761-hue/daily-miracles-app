# 노트북LM 입력 템플릿 (공공용 PPT 전용)

> **사용법**: 각 섹션을 채운 후 노트북LM에 업로드 → 마스터 프롬프트와 함께 실행

---

## 📋 기본 정보

```
[프로젝트 유형]: (관광/행사/용역 중 선택)
[공모명]:
[제출 마감일]:
[발주처]:
```

---

## 1️⃣ 조직 개요

> 회사/조직 소개 (실명 사용 가능 - 익명화 자동 처리됨)

```
[조직명]:
[설립연도]:
[주요 사업 영역]:
[조직 규모]: (예: 상근 인원 N명, 계절 인력 N명)
[지역 기반]:
```

**핵심 역량 (3개 이내):**
1.
2.
3.

---

## 2️⃣ 유사 실적

> 각 실적은 "행사유형 + 규모 + 역할" 형식으로 작성

### 실적 1
```
[행사 유형]: (예: 지역 축제, 학술대회, 기관 워크숍)
[규모]: (예: 참가자 약 500명 규모)
[수행 역할]: (운영 총괄 / 운영 지원 / 연계 협력)
[기간]:
[주요 내용]:
```

### 실적 2
```
[행사 유형]:
[규모]:
[수행 역할]:
[기간]:
[주요 내용]:
```

### 실적 3
```
[행사 유형]:
[규모]:
[수행 역할]:
[기간]:
[주요 내용]:
```

```

|------|--------|------|

```

## 🚀 사용 방법

1. **이 템플릿 복사** → 새 문서에 붙여넣기
2. **각 섹션 채우기** (빈 칸은 삭제해도 됨)
3. **노트북LM에 업로드**
4. **마스터 프롬프트 입력** (`notebooklm-master-prompt.md` 참조)
5. **PPT 생성 요청**

---

## ⚠️ 주의사항

- 실명 사용 OK → 노트북LM이 자동 익명화
- 수치는 "약 N명", "N개 이상" 등 대략적으로 표기
- 과장 표현 자제 (마스터 프롬프트가 필터링하지만 입력부터 주의)
