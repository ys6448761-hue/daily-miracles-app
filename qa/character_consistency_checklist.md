# 캐릭터 일관성 QA 체크리스트

> **용도**: 생성된 이미지의 캐릭터 일관성 평가
> **기준**: 캐릭터당 10점 만점, 6명 합산 60점 만점
> **통과 기준**: 54/60 이상 (90%+) = 초록불

---

## 평가 기준 (캐릭터당 10점)

### 1. 얼굴 특징 (3점)

| 점수 | 기준 |
|------|------|
| 3점 | 레퍼런스와 완전 일치, 스타일 유지 |
| 2점 | 약간의 차이 있으나 인식 가능 |
| 1점 | 눈에 띄는 차이, 다른 캐릭터처럼 보임 |
| 0점 | 완전히 다른 얼굴/스타일 |

### 2. 헤어 실루엣 (2점)

| 점수 | 기준 |
|------|------|
| 2점 | 실루엣 완전 일치 |
| 1점 | 약간의 변형 있으나 인식 가능 |
| 0점 | 다른 헤어스타일 |

### 3. 의상 디테일 (2점)

| 점수 | 기준 |
|------|------|
| 2점 | 의상 디테일 2개 모두 정확 |
| 1점 | 1개만 정확 |
| 0점 | 의상 변경됨 |

### 4. 대표 색감 (2점)

| 점수 | 기준 |
|------|------|
| 2점 | 3가지 대표색 모두 정확 |
| 1점 | 1-2가지만 정확 |
| 0점 | 색상 변경됨 |

### 5. 체형/비율 (1점)

| 점수 | 기준 |
|------|------|
| 1점 | 체형/비율 일치 |
| 0점 | 체형/비율 변경됨 |

## 신호등 판정 기준

### 🟢 초록불 (자동 승인)

```
조건:
- 총점 54/60 이상 (90%+)
- 모든 캐릭터 개별 점수 8/10 이상
- 스타일 일관성 확인됨

액션:
- 즉시 사용 승인
- Golden Prompt 승격 고려
- 메타데이터 저장
```

### 🟡 노란불 (리롤 필요)

```
조건:
- 총점 42-53/60 (70-89%)
- 또는 1명 이상 7/10 미만

액션:
- 리롤 1-2회 시도
- 문제 캐릭터 레퍼런스 재첨부
- "바꾸지 말 것" 명시 후 재생성
- 2회 리롤 후에도 미달 시 → 빨간불
```

### 🔴 빨간불 (수동 개입)

```
조건:
- 총점 42/60 미만 (70% 미만)
- 리롤 2회 후에도 미달
- 심각한 스타일 불일치

액션:
- CRO(재미) 또는 CEO(푸르미르) 보고
- 프롬프트 전면 재검토
- 레퍼런스 이미지 교체 고려
- 수동 편집 또는 합성 고려
```

## 빠른 체크 (Quick Check)

### 5초 검사 항목

```
□ 색상이 맞는가? (가장 눈에 띄는 오류)
□ 캐릭터를 식별할 수 있는가?
□ 스타일이 일관되는가? (수채화/지브리)
□ 3D/실사 느낌이 나는가? (→ 실패)
□ 텍스트/워터마크가 있는가? (→ 실패)
```

### 통과 시 추가 확인

```
□ 세부 디테일 (의상, 상징물)
□ 포즈/배치 적절성
□ 배경 일관성
□ 해상도/품질
```

## 자주 발생하는 문제 & 해결

| 문제 | 원인 | 해결책 |
|------|------|--------|
| 색상 변경됨 | 레퍼런스 미첨부 | 캐릭터 레퍼런스 필수 첨부 |
| 얼굴이 상세함 | Negative 누락 | "minimal face, NOT detailed" 강조 |
| 3D 느낌 | 스타일 가드 누락 | consistency_guard.md 전문 삽입 |
| 캐릭터 혼동 | 설명 불충분 | 캐릭터별 요약 명시 |
| 의상 변경됨 | 바이블 미참조 | 의상 디테일 2개 명시 |
