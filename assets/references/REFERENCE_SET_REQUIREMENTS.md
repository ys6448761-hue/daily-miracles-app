## 레퍼런스 이미지 규격

### 01_front.png (정면)
- 각도: 정면 또는 약간 틀어진 정면
- 범위: 상반신 (가슴 위)
- 용도: 주 식별용, 얼굴/헤어/의상 시그니처 확인

### 02_3quarter.png (3/4 각도)
- 각도: 45도 회전
- 범위: 상반신
- 용도: 입체감, 의상 측면 확인

### 03_fullbody.png (전신)
- 각도: 정면 또는 3/4
- 범위: 전신 (발끝까지)
- 용도: 체형, 비율, 전체 의상 확인

### master_sheet.png (권장)
- 내용: 표정 3-5개, 포즈 2-3개
- 용도: 다양한 상황에서 일관성 보장
