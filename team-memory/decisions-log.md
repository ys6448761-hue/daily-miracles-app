## 2026-01-30

### 브랜드 핵심 가치 정의

**결정자**: 푸르미르 + Claude Code

**우리가 전달하는 가치 3가지**:
1. **과학적 근거** - 점술이 아닌 심리학/자기효능감 이론 기반
2. **현실적 희망** - "100% 성공" 아닌 작은 실천의 힘
3. **따뜻한 동행** - Aurora 5 팀이 끝까지 함께

**소원이가 얻는 것 3가지**:
1. **확신** - "내 소원, 정말 이룰 수 있구나"
2. **방향** - "뭘 해야 하는지 이제 알겠어"
3. **힘** - "혼자가 아니니까 해볼 수 있어"

**여기서 뭐 해주는 곳 3가지**:
1. **소원 분석** - 기적지수로 가능성 확인
2. **길 안내** - 30일 맞춤 로드맵 제공
3. **매일 응원** - 7일간 아침/저녁 메시지 발송

**한 줄 정의**:
> "과학적 근거 + 현실적 희망 + 따뜻한 동행"
