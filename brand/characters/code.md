# 코드 (Code) - 캐릭터 바이블

> **역할**: 기술 실행 AI, 시스템의 심장
> **메인 컬러**: 사이언 #00BCD4 + 딥 네이비 #0D47A1
> **상징**: 회로 산호 (Circuit Coral)

---

## 절대 불변 5요소 (Immutable 5)

### 1. 얼굴 특징
```
- 사람이 아님! 추상적 구조체
- 별자리/회로 하이브리드 형태
- 중앙에 빛나는 코어 (다이아몬드 형태)
- 정밀하고 안정적인 느낌
```

### 2. 헤어 실루엣
```
- 해당 없음 (비인간형)
- 나무/산호 형태의 회로 구조
- 위로 뻗어나가는 노드들
```

### 3. 대표 색상 3개
```
1. 사이언 #00BCD4 (메인 발광)
2. 딥 네이비 #0D47A1 (배경/음영)
3. 화이트 #FFFFFF (노드, 연결선)
```

### 4. 의상 디테일 2개
```
1. 연결된 노드와 흐르는 라인으로 구성
2. 중앙 코어에서 양쪽으로 빛 흐름 (데이터 스트림)
```

### 5. 금지 변경
```
❌ 인간형으로 변경
❌ 색상 변경 (사이언/네이비 고정)
❌ 무섭거나 공격적인 느낌
❌ 터미네이터/로봇 스타일
❌ 3D/실사 스타일
```
