# memory/

> 학습된 패턴과 기억 저장소

---

## 접근 규칙

맥락에 따라 자동 참조. 패턴 학습/조회 작업 시 활성화.

---

## 포함 대상

| 폴더 | 내용 |
|------|------|
| `prompts/` | 효과적인 프롬프트 패턴 |
| `scenes/` | 재사용 가능한 장면 기억 |
| `emotions/` | 감정 표현 패턴 |
| `failures/` | 실패 기록 — **삭제 금지** |
| `successful-patterns/` | 성공한 접근 패턴 |
| `growth-history/` | 시간 순 성장 기록 |

---

## 원칙

- `failures/`는 지우지 않는다. 반복 방지를 위한 기억이다
- 성공과 실패를 같은 무게로 기록한다
