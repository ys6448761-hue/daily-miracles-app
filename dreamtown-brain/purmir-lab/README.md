# purmir-lab/

> 푸르미르 개인 사고 공간

---

## 접근 규칙

**자동 참조하지 않는다.**

이 폴더는 OpenClaw가 기본값으로 읽지 않는다.
탐색은 명시 요청 시에만 한다.

```
명시 요청 예시:
"purmir-lab/dream-seeds 탐색해줘"
"raw-thoughts에서 관련 메모 찾아줘"
```

---

## 포함 대상

| 폴더 | 내용 |
|------|------|
| `raw-thoughts/` | 날것의 생각. 형식 없음 |
| `dream-seeds/` | 아이디어 씨앗. 완성 불필요 |
| `emotional-notes/` | 감정 메모. 논리 불필요 |
| `future-worlds/` | 미래 세계관 스케치 |
| `experiments/` | 실험 기록. 실패해도 OK |
| `unresolved/` | 미해결 질문. 답 없어도 기록 |

---

## 사용 원칙

- 완성도를 요구하지 않는다. 쓰다 멈춰도 된다
- 여기 있는 내용은 초안이다. `official/`과 섞지 않는다
- `official/`로 이동은 CEO 검토 + 승인 후에만
- 삭제보다 `unresolved/`로 이동을 우선한다

---

## 이 폴더에 넣지 않는 것

- 승인된 확정 규칙 → `official/`
- 학습된 패턴 → `memory/`
