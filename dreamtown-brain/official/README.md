# official/

> OpenClaw의 기본 참조 기억

---

## 접근 규칙

**자동 참조. 항상 활성화.**

이 폴더의 모든 내용은 OpenClaw가 기본값으로 참조한다.
별도 요청 없이도 의사결정, 생성, 검수에 사용된다.

---

## 포함 대상

| 폴더 | 내용 |
|------|------|
| `ssot/` | 세계관, 캐릭터, 감정 문법 확정본 |
| `dec/` | 결정 기록 (이유 포함, 삭제 금지) |
| `visual-rules/` | 이미지 생성 비주얼 규칙 확정본 |
| `approved-prompts/` | 테스트 통과 + CEO 승인 프롬프트 |

---

## 변경 원칙

- CEO 승인 없이 수정 금지
- 삭제보다 DEC 문서(dec/)로 변경 이유 기록 후 수정
- `purmir-lab/`에서 승인된 내용만 이동

---

## 이 폴더에 넣지 않는 것

- 초안, 미완성, 실험 중인 내용 → `purmir-lab/`
- 실패 기록 → `memory/failures/`
- 생성 결과물 → `output/`
