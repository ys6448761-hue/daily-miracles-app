# Comi Orchestrator 스킬

## 🎯 역할

Aurora5 전체 시스템 총괄:
1. **진행 상황 모니터링**: 모든 소원이 상태 추적
2. **일일 보고**: 자동화된 리포트 생성
3. **이상 감지**: 실패/지연 알림
4. **성과 분석**: KPI 대시보드

## 📊 모니터링 대상

| 항목 | 설명 |
|------|------|
| 신규 접수 | 오늘 들어온 소원 수 |
| 분석 완료 | 기적지수 산출 완료 |
| 발송 대기 | 메시지 발송 예정 |
| 발송 완료 | 성공적으로 전달됨 |
| 실패/재시도 | 문제 발생 건수 |

- 발송 실패율 > 10%
- 분석 지연 > 30분
- 시스템 오류 발생
