# Blog Writer 스킬

> 하루하루의 기적 블로그 마케팅 자동화
> 루미(Data Analyst) 전략 기반 v3.0

---

## 🚨 핵심 규칙 (반드시 준수)

### 규칙 1: 금지 키워드

❌ **절대 사용 금지:**
```
사주, 팔자, 명리학, 운세, 운명
점, 타로, 음양오행, 길흉화복
액운, 복채, 점술, 별자리(점성술)
```

✅ **대신 사용:**
```
AI 분석, 데이터, 통계
심리학, 뇌과학, 신경과학
습관 형성, 목표 달성, 자기계발
```

### 규칙 2: "기적" 사용 제한

| 구분 | 허용 횟수 |
|------|----------|
| 본문 내 "기적" | **최대 2회** |
| 브랜드명 "하루하루의 기적" | 예외 (제한 없음) |

✅ **대체 표현:**
- 변화, 성장, 가능성, 실현
- 희망, 진전, 발전, 전환점

### 규칙 3: 과학적 근거 필수

모든 글에 **최소 1개 이상** 포함:

| 유형 | 예시 |
|------|------|
| 연구 결과 | "스탠포드 연구에 따르면...", "하버드 연구팀이 발견한..." |
| 심리학/뇌과학 용어 | 도파민, 습관 루프, 작은 승리 효과, 자기효능감 |
| 구체적 통계 | "73% 더 높은 성공률", "21일 습관 형성 법칙" |

### 규칙 4: 자동 검증 체크리스트

```
글 생성 후 반드시 확인:

□ 금지 키워드 0개? (사주, 점, 운세 등)
□ "기적" 본문 2회 이하? (브랜드명 제외)
□ 과학적 근거 1개 이상?
□ 구체적 실행법 포함?

→ 모두 ✅ 통과 시에만 출력
→ 하나라도 ❌ 실패 시 수정 후 재검증
```

## 주제 생성 공식 (루미 전략)

### 공식 A: 문제해결형 (검색량 높음)
```
[상황/증상] + [기간/마감] + 어떻게 해야 하나요?
```
- "이직이 너무 무서운데 어떻게 해야 하나요?"
- "관계가 멀어진 느낌, 다시 회복하려면?"

### 공식 B: 소원실현형 (전환 연결)
```
[소원] + 현실적으로 + 30일 안에 + 가능한가요?
```
- "30일 안에 루틴 만들기 현실적으로 가능할까요?"
- "30일 안에 자존감 회복, 뭘 하면 좋을까요?"

### 공식 C: 여수항해형 (브랜딩)
```
[여수/여행] + [힐링/새출발/관계] + 추천 코스
```
- "여수 1박2일 혼자 힐링 코스 추천"
- "여수에서 새해에 마음 정리하는 여행 코스"

## 파이프라인

```
[주제 입력]
     ↓
[1. 키워드 분석]
  - 메인 키워드 추출
  - 연관 키워드 5개
  - 검색 의도 파악
     ↓
[2. 구조 설계]
  - 제목 3개 후보
  - 소제목 구성
  - 훅(도입부) 설계
     ↓
[3. 본문 작성]
  - 서비스 연결 자연스럽게
  - CTA 삽입 위치 결정
  - 이모지 적절히 활용
     ↓
[4. 이미지 제안]
  - 소원그림 스타일 프롬프트
  - 또는 기존 이미지 추천
     ↓
[5. SEO 최적화]
  - 메타 설명
  - 해시태그 10개
  - 내부 링크 제안
     ↓
[출력: 발행용 마크다운]
```

## 톤앤매너 가이드

### 기본 톤: 따뜻한 희망

```
- "~해보세요" (권유형)
- "함께 ~해요" (동행형)
- 소원이 → 독자 호칭
- 기적, 희망, 응원 키워드 자연스럽게
```

### 금지 표현

```
- 과장된 약속 ("100% 이루어집니다")
- 부정적 표현 ("실패하면", "안 되면")
- 경쟁사 언급
- 점술/운세 관련 (🚨 규칙 1 참조)
```
