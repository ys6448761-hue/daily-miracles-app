# 배치 처리 가이드라인

## 언제 배치로?

### 배치 처리 적합:
- 동일 유형 작업 10개 이상
- 각 작업이 독립적
- 순서 무관
- 실패해도 다른 작업에 영향 없음

### 배치 처리 부적합:
- 작업 간 의존성 있음
- 순서가 중요함
- 실시간 응답 필요

## 배치 크기

| 작업 수 | 배치 크기 | 병렬도 |
|---------|----------|--------|
| 1-10 | 전체 1개 | 1 |
| 11-50 | 5개씩 | 2-10 |
| 51-200 | 10개씩 | 5-20 |
| 200+ | 20개씩 | 10+ |

## 에러 처리

```
배치 실행 중 일부 실패 시:
1. 성공한 작업은 그대로 진행
2. 실패한 작업만 별도 배치로 재시도
3. 3회 재시도 후에도 실패 → 코미 알림
4. 전체 결과에 실패 목록 포함
```
