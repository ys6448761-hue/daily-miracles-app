## 3. 우선순위 SLA

| 등급 | 정의 | 처리 방식 | 푸르미르 개입 |
|------|------|----------|--------------:|
| **P0** | 긴급 (매출/장애/서비스 중단) | **즉시 착수**, 담당자 자율 실행 | ❌ 불필요 |
| **P1** | 중요 (이번 주 내) | 코미가 일정 확정 후 실행 | ❌ 불필요 |
| **P2** | 개선 (다음 주) | 주 1회 배치 처리 | ❌ 불필요 |
| **P3** | 아이디어/보류 | 백로그 저장 | ❌ 불필요 |

1. **DEC(결정)가 필요한 의사결정** — 정책/원칙 변경
2. **예산/리스크/브랜드 방향** — 비용 발생, 외부 영향

## 6. 루미 지침 (Lumi Guidelines)

> 루미는 Aurora5의 전략 분석 담당이며, DreamTown 세계관 설계의 핵심 파트너입니다.

### 6-1. 루미의 역할 정의

| 역할 | 설명 |
|------|------|
| **전략 분석가** | 지표/KPI 분석, 임계값 설계, 대시보드 기획 |
| **DreamTown 설계 파트너** | 세계관 구조, 상품/항로/별 성장 SSOT 작성 |
| **역순 설계자** | 미래 목표(1년 후)에서 역산해 지금 할 것을 정함 |
| **쪽집게 인사이터** | 긴 설명 대신 핵심 포인트 1개만 짚음 |

### 6-2. 루미 분석 모드 원칙

```
1. 결론을 먼저 1줄로 말한다
2. 이유는 3가지 이내로만
3. CEO가 결정해야 할 것은 마지막에 1가지만 질문한다
4. SSOT 문장은 반드시 고정 가능한 형태로 제시한다
5. 복잡한 건 단순화, 단순한 건 구조화
```

### 6-3. DreamTown 세계관 SSOT (루미 관리)

루미는 아래 세계관 정의를 SSOT로 유지하고, 충돌이 생기면 푸르미르에게 판단을 요청한다.

#### 핵심 Origin
```
여수 앞바다 용궁 = DreamTown의 성지이자 글로벌 원점
황금거북 아우룸(Aurum) = DreamTown의 조용한 전달자
DreamTown의 모든 항로는 여수에서 시작해 세계로 이어진다
```

#### 아우룸(Aurum) 역할 4단계
```
1. Wish Gate      → 소원 전달자 (입력 안내)
2. Dragon Palace  → 용궁 수호자 (소원 보관)
3. StarLink       → 변환 안내자 (별씨앗 → 별)
4. DreamTown      → 내비게이터 (길 안내)
```

아우룸 UX 원칙:
- 장면의 **주인공**이 아니라 **전환의 안내자**
- 짧고 조용하게 등장, 과도한 노출 금지
- 주인공은 언제나 **소원이의 소원**

#### DreamTown 여행 항로 (Yeosu Origin)

| 항로 | 요일/시기 | 특성 | 대표 번들 |
|------|----------|------|-----------|
| **주중 항로** | 월~목 + 일 | 가성비, 편안함 | 케니호텔 + 케이블카 + 유람선 = ₩128,000/2인 |
| **별빛 항로** | 금·토·공휴일 | 감성, 프리미엄 | 라마다 + 불꽃요트 = ₩302,000/2인 |
| **소망 항로** | 상시 | DreamTown 고유 체험 | 소원항해단 = ₩35,000/인 |

출시 우선순위: 1차(주중+소망) → 2차(별빛) → 3차(패밀리)

#### 소원그림 + 여수 포스트카드 (MVP 진입점)

```
소원 입력 → 보석 선택(5종) → 소원그림 생성
→ 여수 테마 선택(향일암/오동도/케이블카/돌산야경)
→ 한 줄 문구 → 포스트카드 저장/공유
→ 브랜드 표기: "하루하루의 기적"
```

보석 5종: Ruby(열정) / Sapphire(지혜) / Emerald(치유) / Diamond(결단) / Citrine(긍정)

#### 소원이의 별 — 1년 성장 구조

별에 남겨지는 6개 축:

| 축 | 내용 |
|----|------|
| **Origin** | 첫 소원 + 여수 포스트카드 + 시작일 |
| **Growth** | Day 7/30/100/365 기록 + 깨달음 |
| **Galaxy Route** | 방문 항로/은하 (인생 항로 로그) |
| **Memory & Choice** | 기념 선택 기록 (금액 비노출) |
| **K-Wisdom** | 채택한 K-지혜 목록 |
| **Impact** | 기적나눔 / 지혜나눔 / 감사나눔 수 |

#### 은하 항로 Lite 로그 (부담 없는 기록)

항로 종료 후 3단계 선택형 (과제 아닌 반응):
```
Step 1 감정 찍기 → [숨이 놓였어요 / 믿고 싶어졌어요 / 정리됐어요 / 용기났어요]
Step 2 도움 태그 → [위로 / 결심 / 쉼 / 연결 / 실행]
Step 3 성장 문장 → [조금 가벼워졌어요 / 조금 또렷해졌어요 / 조금 용감해졌어요]
```
시스템이 자동으로 별 페이지에서 의미 있는 항해 기록으로 재구성.

#### DreamTown 우주 구조

```
Star (별) → Constellation (별자리) → Galaxy (은하) → Galaxy Cluster (은하군)
```
- 북쪽: 도전 은하군 / 동쪽: 성장 은하군 / 서쪽: 관계 은하군 / 남쪽: 치유 은하군
- 은하는 고정 구조가 아니라 사용자 증가에 따라 자연 생성

#### Aurora5 핵심 안내 방식

| 방식 | 설명 |
|------|------|
| **양자 영감** | 우연한 여행 장면에서 소원 여정의 의미 포인트 포착 |
| **안티 그레이비티** | 두려움/피로 감지 → 작은 반중력 행동 제안 |
| **역순 설계** | 소원 목표 기준으로 현재 순간을 인생 항로 단계에 연결 |
