# 브랜드 에센스 정의 & 팀 메모리 시스템 구축

> 대화 일시: 2025-01-30 오전

---

## 요약

브랜드 핵심 가치 3가지를 정의하고, 팀 기억 시스템(.claude/team-memory/)을 구축했으며, Google Drive 백업 스크립트를 완성했다.

---

## 주요 결정사항

- 브랜드 핵심 가치 3가지: **과학적 근거 + 현실적 희망 + 따뜻한 동행**
- 소원이가 얻는 것 3가지: **확신, 방향, 힘**
- 사이트가 뭐 하는 곳: **소원 분석, 길 안내, 매일 응원**
- 기억 시스템: GitHub 기반 단순 구조 (.claude/team-memory/)
- 백업: Google Drive 수동 백업 (npm run backup-memory)

---

## 생성된 파일

### 브랜드
- `skills/design-system/brand-essence.md` - 브랜드 에센스 문서
- `output/brand-essence.pdf` - PDF 버전
- `scripts/generate-brand-essence-pdf.js` - PDF 생성 스크립트

### 팀 메모리
- `.claude/team-memory/context.md` - 현재 맥락
- `.claude/team-memory/decisions.md` - 주요 결정사항
- `.claude/team-memory/learnings.md` - 배운 것들

### 백업 시스템
- `scripts/backup-memory.js` - Google Drive 백업 스크립트
- `scripts/save-conversation.js` - 대화 저장 스크립트
- `team-memory/decisions-log.md` - 결정 로그 (루트)

---

## 배운 것들

- Google Drive Desktop이 G:\내 드라이브에 마운트되어 있음
- Drive 연동보다 GitHub 직접 저장이 더 단순하고 효과적
- 세션 간 연속성 유지를 위해 context.md가 핵심
