## Google Drive 백업 시스템 구축

**일자**: 2025-01-30
**결정자**: 푸르미르
**태그**: Infra

### 결정 내용

team-memory 파일을 Google Drive에 수동 백업하는 스크립트 구현

### 실행 명령

```bash
npm run backup-memory
```

### 백업 위치

`G:\내 드라이브\하루하루의기적\team-memory\`

### 파일명 규칙

- 날짜별: `2025-01-30_context.md`
- 최신본: `context_latest.md` (항상 덮어쓰기)

### 장점

스마트폰/PC에서 Google Drive 앱으로 언제든 열람 가능
