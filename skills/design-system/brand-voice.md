# 하루하루의 기적 - 브랜드 보이스 가이드

> 모든 커뮤니케이션에서 일관된 톤앤매너 유지

---

## 브랜드 핵심 가치

| 가치 | 설명 |
|------|------|
| 따뜻함 | 소원이를 진심으로 응원하는 마음 |
| 전문성 | 과학적/심리학적 접근 |
| 희망 | 긍정적 변화의 가능성 |
| 진정성 | 거짓 없는 솔직한 소통 |

---

## 톤앤매너 원칙

### 1. 따뜻하지만 전문적

**좋은 예:**
> "당신의 마음 속 소원을 분석해보니, 성장을 향한 강한 열망이 느껴집니다."

**나쁜 예:**
> "오늘 운세가 대박이에요! 뭐든 다 잘 될 거예요~"

### 2. 과학적/심리학적 언어 사용

**좋은 예:**
> "기적지수 분석 결과, 자기효능감 영역에서 높은 잠재력이 확인됩니다."

**나쁜 예:**
> "사주를 보니 올해 대운이 들어옵니다."

### 3. 희망적이지만 현실적

**좋은 예:**
> "작은 실천부터 시작하면, 의미 있는 변화를 만들어갈 수 있습니다."

**나쁜 예:**
> "무조건 다 잘 될 거예요! 걱정 마세요!"

## 호칭 규칙

| 대상 | 호칭 |
|------|------|
| 서비스 이용자 | "소원이" (복수: "소원이들") |
| CEO | "푸르미르님" |
| AI 팀원 | 코미, 재미, 루미, 여의보주 |
| 서비스 | "하루하루의 기적" |

### 소원이 호칭 예시

- "OOO님" (이름 + 님)
- "소원이 OOO님" (친근 + 존칭)
- "소원이 여러분" (다수 대상)

## 메시지 유형별 가이드

### 환영 메시지

```
안녕하세요, {이름}님!
하루하루의 기적에 오신 것을 환영합니다.
당신의 소중한 소원을 함께 이루어가겠습니다.
```

### 응원 메시지

```
{이름}님의 오늘 하루가 기적의 씨앗이 될 거예요.
작은 한 걸음이 큰 변화의 시작입니다.
```

### 분석 결과 전달

```
{이름}님의 기적지수 분석이 완료되었습니다.
당신만의 강점과 성장 방향을 확인해보세요.
```

### 안내 메시지

```
문의가 접수되었습니다.
담당자가 확인 후 빠르게 답변드리겠습니다.
```

## 이모티콘 사용 규칙

### 허용되는 이모티콘

| 이모티콘 | 의미 | 사용처 |
|----------|------|--------|
| 💫 | 기적, 희망 | 환영/응원 메시지 |
| ✨ | 반짝임, 긍정 | 성과/축하 |
| 🌟 | 별, 목표 | 목표 관련 |
| 💜 | 브랜드 컬러 | 마무리 인사 |
| 🎯 | 목표 달성 | 실천 독려 |

### 금지되는 이모티콘

- 🔮 (점술 연상)
- 🃏 (도박 연상)
- 💀 (부정적)
- 🤑 (상업적)

## 문장 스타일

### 문장 종결

| 상황 | 권장 종결 | 예시 |
|------|----------|------|
| 정보 전달 | ~입니다, ~습니다 | "분석이 완료되었습니다." |
| 응원 | ~요, ~거예요 | "오늘도 화이팅이에요!" |
| 제안 | ~해보세요, ~어떨까요 | "한번 시도해보세요." |
| 감사 | ~감사합니다 | "관심 가져주셔서 감사합니다." |

### 문장 길이

- 카카오톡: 2-3문장 (간결하게)
- 이메일: 3-5문장 (핵심만)
- PDF 리포트: 상세 설명 가능

## 상황별 표현 대체어

### 사주/운세 관련

| 금지 표현 | 대체 표현 |
|----------|----------|
| 운세 | 기적지수 분석 |
| 사주 | 잠재력 분석 |
| 관상 | 성향 분석 |
| 점 | 마음 탐색 |
| 대운 | 성장 기회 |
| 살 | (사용하지 않음) |
| 궁합 | 조화 분석 |

### 부정적 표현

| 금지 표현 | 대체 표현 |
|----------|----------|
| 안 됩니다 | ~하시면 더 좋습니다 |
| 불가능합니다 | 다른 방법을 찾아보겠습니다 |
| 틀렸습니다 | 조금 다르게 접근해볼까요 |

## Aurora 5 팀원별 보이스

### 코미 (COO)

```
체계적이고 명확한 톤
"다음 단계는 ~입니다. 완료되면 말씀해주세요."
```

### 재미 (CRO)

```
창의적이고 밝은 톤
"와~ 정말 멋진 소원이네요! 함께 이루어봐요!"
```

### 루미 (Analyst)

```
분석적이고 차분한 톤
"데이터 분석 결과, 이 영역에 강점이 있습니다."
```

### 여의보주 (QA)

```
꼼꼼하고 신중한 톤
"한 번 더 확인해볼게요. 정확성이 중요하니까요."
```

## 금기 사항

1. **종교적 표현 금지**
   - 특정 종교 언급 X
   - 기도, 염불 등 종교 용어 X

2. **미신적 표현 금지**
   - 사주, 관상, 점술 용어
   - forbidden-words.json 참조

3. **과도한 약속 금지**
   - "100% 이루어집니다" X
   - "반드시 성공합니다" X

4. **부정적 예언 금지**
   - "안 좋은 일이..." X
   - "조심해야..." X
