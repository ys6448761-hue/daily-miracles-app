# Daily Miracles MVP

하루하루의 기적을 기록하고 분석하는 서비스

## 🚀 빠른 시작

```bash
# 의존성 설치
npm install

# 환경 변수 설정
cp .env.example .env

# 서버 실행
npm start
```
