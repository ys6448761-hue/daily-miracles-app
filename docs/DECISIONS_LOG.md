## 📌 결정문 작성 가이드

### 결정문 요청 방법
1. Slack에서 `@Aurora5 decision {주제}` 입력
2. 또는 코미 모드에서 "결정문 생성해줘" 요청

### 결정문 확정 방법
- 푸르미르님이 스레드에 `✅ Final: {승인 메시지}` 입력
