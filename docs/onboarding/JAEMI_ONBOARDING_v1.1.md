### 5.2 스타일 앵커 1장 선정

**저장 경로**: `assets/references/style/miracle_watercolor_anchor.png`

**스타일 앵커란?**
```
전 매체(단체샷/웹툰/영상)에 공통 적용할 "톤 기준 이미지" 1장
- 나노바나나가 이 이미지를 스타일 레퍼런스로 참조
- 모든 결과물의 색감/분위기/터치가 이 앵커에 맞춰짐
- 한번 확정되면 10회 검증 프로토콜 트리거됨
```

**선정 기준**:
1. 지브리 + 웹툰 수채화 스타일
2. 따뜻하고 희망적인 색감
3. 캐릭터 6명의 메인 컬러와 조화

**선정 프로세스**:
```
1. 재미가 후보 이미지 2-3장 선별
2. 팀 리뷰 (코미 + 여의보주)
3. 최종 1장 확정 → 앵커 경로에 저장
4. 10회 검증 프로토콜 자동 트리거
```
