# 최수민방식 치트시트

빠른 참조용 핵심 요약

---

## 컨텍스트 절약

| 잘못된 방식 | 올바른 방식 |
|------------|------------|
| DB 전체 로드 | 스크립트로 추출 |
| 100개 한꺼번에 | 10개씩 배치 |
| 긴 결과 출력 | 파일로 저장 |
| 파일 반복 읽기 | 한번 읽고 재사용 |

---

## 피드백 반영

```
실수 발견
    ↓
에이전트 .md 열기
    ↓
## 지침 섹션에 추가
    ↓
AI가 즉시 학습
```

---

## 배치 처리 패턴

```
"[목록]에서 미완료 항목을 최대 10개씩 처리하고 반복하라"
```

---

## 프롬프트 → 에이전트

```
prompts/ 저장
    ↓
{{변수}} 추가
    ↓
3회 이상 사용
    ↓
agents/ 승격
```

---

## 에이전트 필수 섹션

```markdown
## 지침
[피드백 반영 작업 지침]

## 자가 검증
[완료 후 체크리스트]

## 피드백 기록
| 날짜 | 문제 | 해결 | 지침 추가 |
```

---

## 자가 검증 4단계

1. **도그푸딩** - 직접 써봐
2. **검증 로직** - 스킬에 체크 포함
3. **지침 보강** - 즉시 반영
4. **최종 종합** - 파이프라인 끝 검증
