## 1. Core Philosophy

### 핵심 선언

> **"당신의 소원은 혼자가 아닙니다."**

DreamTown은 이 하나의 믿음에서 시작된다.

모든 인간은 소원을, 희망을, 두려움을, 완성되지 않은 꿈을 품고 있다.
DreamTown은 그 소원들이 **별이 되어 서로를 비추는 하늘**이다.

### 존재 이유

- 소원이 **보관되고 기억되는** 공간
- 소원이 **별이 되어 공개되는** 하늘
- 별들이 **서로를 비추는** 공동체

DreamTown은 SNS가 아니다. **소원이 자라는 하늘**이다.

### 공식 오프닝 문장 (CEO 확정)

> "여수 바다에서 시작된 하늘."
> "당신의 소원은 혼자가 아닙니다."

### 세계관 (Worldview) — 소원항해

```
소원 (Wish)         ← 시작점
    ↓
항해 (Journey)      ← 소원이 → 소망이로 성장
    ↓
깨달음 (Realization) ← 여정에서 얻은 지혜
    ↓
나눔 (Sharing)      ← 지혜를 별자리로 나누기
```

### 절대 금지

DreamTown은 **절대** 다음을 하지 않는다:
- 사주, 점술, 관상, 운세, 대운, 궁합 언급
- 미래 예측, 의료 조언, 법률 조언
- 사용자 감정 평가 또는 판단

## 2. Universe Structure

### 세계 수직 구조

```
현실 세계 (온라인 + 오프라인)
        ↓
   여수 바다 (Origin — 출발점)
        ↓
   빛구슬 (Light Orb) — 소원의 형태화
        ↓
   별씨앗 (Star Seed) — 숙성 단계
        ↓
   디지털 용궁 (Dragon Palace) — 소원 원본 보관
        ↓
   아우룸 (Aurum) — 소원의 전달자
        ↓
   Aurora5 — 동행 시스템
        ↓
   StarLink = Aurora Path — 빛의 길
        ↓
   DreamTown 하늘 — 별 / 연결 / 이야기
        ↓
   별 → 별자리 → 은하 → 은하군
```

### Golden Turtle Axis (세계관 핵심 축)

```
여수 바다
    ↓
황금 거북 별자리 (Golden Nine Constellation)
    ↓
DreamTown 은하
```

**바다 → 하늘 → 우주** — DreamTown의 수직 축이자 브랜드 상징.

### 세계관 원칙

| 원칙 | 내용 |
|------|------|
| 용궁이 중심 | 모든 소원은 용궁으로 수렴된다 |
| 소원은 사라지지 않는다 | 빛구슬로 용궁에 영구 보관 |
| 성장은 혼자가 아니다 | Aurora5가 여정을 동행 |
| 기적은 과학이다 | 심리학·뇌과학 기반, 사주/점술 배제 |
| 여수는 출발점 | 현실 공간과 디지털이 여수에서 만남 |

## 3. Characters

### 소원이 (Sowoni)

- **역할**: 소원을 품은 모든 사람을 대표하는 내러티브 캐릭터
- **의미**: "소원이" = 소원을 가진 사람 그 자체
- **형태**: 20~22세 한국 대학생 / 웹툰 스타일 2D 캐릭터
- **서사 역할**: 사용자가 자신을 투영하는 주인공. 아우룸과 함께 Dragon Palace → DreamTown Star 여정을 걷는다.

### 아우룸 (Aurum)

**공식 정의**:
> 아우룸은 디지털 용궁의 황금 거북 수호자이자, 인간의 소원을 바다와 하늘 사이로 인도하는 DreamTown의 내비게이터입니다. 온라인의 빛구슬과 오프라인의 종이비행기를 용궁으로 전달하고, 잠든 소원이 별씨앗에서 별로 떠오를 때까지 조용히 지켜봅니다.

| 역할 | 장면 | 감정 |
|------|------|------|
| 소원 전달자 | Wish Gate | 시작 / 기대 |
| 용궁 수호자 | Dragon Palace | 안심 / 기다림 |
| StarLink 안내자 | 별 탄생 순간 | 감동 / 변환 |
| DreamTown 내비게이터 | 탐험 중 | 연결 / 발견 |

**UX 원칙**: 귀엽지만 가볍지 않음 / 따뜻하지만 과하지 않음 / 안내하지만 주인공은 아님

### 소망이 (Somangi)

- 소원이 여정을 마친 후 도달하는 **상태**
- "자신 안의 빛을 발견하고 다른 이들의 길을 밝히는 사람"
- 목적지가 아닌 **되어가는 과정**

## 7. Product Canon

### Core Loop

```
소원 (Wish)
    ↓
별 탄생 (Star Birth)
    ↓
내 별 (My Star)
    ↓
다시 보기 (Return)
    ↓
새 소원 (New Wish)
    ↓ (반복)
```

> DreamTown은 소원을 별로 남기고 다시 그 별을 보러 돌아오는 루틴형 위로 서비스이다.

### 소원 여정 단계

```
Human Wish → Light Orb → Dragon Palace → Star Seed → StarLink → DreamTown Star
```

### 4대 은하 시스템

| 은하 | 코드 | 방향 | 보석 |
|------|------|------|------|
| 도전 은하 | challenge | 북 | ruby |
| 성장 은하 | growth | 동 | sapphire, diamond |
| 관계 은하 | relationship | 서 | citrine |
| 치유 은하 | healing | 남 | emerald |

### 별 이름 시스템

- NATURE_WORDS(50) × ACTION_WORDS(40) = **2,000가지** 고유 조합
- wish_id UUID → 결정론적 index → 별 이름 부여
- 한국어 받침 규칙 자동 적용 (을/를)

예시: 새벽을 기다리는 별 / 달빛을 품은 별 / 은하를 향하는 별
