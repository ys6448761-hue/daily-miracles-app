## 핵심 구조

DreamTown은 두 개의 세계가 연결된 구조다.

```
현실 세계 (Origin World)
    ↓
여수 바다 (Wish Ocean)
    ↓
Golden Turtle Field (황금 거북 영역)
    ↓
Star Birth Gate (별 탄생 관문)
    ↓
DreamTown 하늘 (Sky Layer)
    ↓
DreamTown 은하 (Galaxy Layer)
```

> **현실 → 상징 → 우주**로 확장되는 구조

---

## DreamTown Axis (공식 세계 축)

```
여수 바다
    ↓
황금 거북 별자리 (Golden Turtle Constellation)
    ↓
DreamTown 은하
```

이 축이 **DreamTown Axis** — DreamTown 세계관 전체의 중심선.

---

## Layer 1 — Origin World (현실 지도)

여수 = DreamTown **Origin City**

### 여수 핵심 장소 × 세계관 대응

| 현실 장소 | 세계관 역할 | 의미 |
|----------|-----------|------|
| 여수 밤바다 | Wish Ocean | 소원의 탄생 바다 |
| 향일암 | Star Birth Gate / Wish Gate | 별이 탄생하는 관문 |
| 오동도 | Wish Harbor | 소원이 모이는 항구 |
| 돌산대교 | Star Bridge | 현실과 DreamTown을 잇는 다리 |

---

## Layer 2 — Golden Turtle Field

여수 주변 실제 지형 = 황금 거북 별자리의 지형 기반

```
백도 / 거문도 / 사도 / 금오도
향일암 / 오동도 / 제석산 / 신풍
```

이 지형이 **Golden Turtle Field** — DreamTown 세계관의 현실 뿌리.

> 창세 신화 연결: `docs/ssot/DreamTown_Origin_Myth_SSOT.md`

## Tourism Layer — 현실 여행 × 세계관 의미

DreamTown은 실제 관광과 연결된다.

| 현실 체험 | 세계관 의미 |
|---------|-----------|
| 여수 밤바다 걷기 | Star Birth Scene 체험 |
| 케이블카 탑승 | Sky Journey — 하늘로 올라가는 항해 |
| 유람선 탑승 | Ocean Voyage — 아우룸의 바다 탐험 |
| 향일암 방문 | Wish Gate — 소원을 올리는 성지 |
| 소원엽서 작성 | 오프라인 소원 → 용궁 저장 |
| 종이비행기 날리기 | 소원을 아우룸에게 맡기는 행위 |

---

## 탐험 흐름 (현실 + 앱 통합)

```
여수 방문 (현실)
    ↓
소원 입력 (앱 또는 오프라인)
    ↓
별 생성 (DreamTown 하늘)
    ↓
별자리 발견
    ↓
은하 탐험
    ↓
다른 소원이의 이야기 발견
```

DreamTown = **여행 + 세계관 + 커뮤니티 플랫폼**
