## DreamTown 5 Layers

```
Layer 1  Origin      여수 바다 → Golden Turtle → DreamTown 하늘
Layer 2  Universe    소원 → 별씨 → 별 → 별자리 → 은하 → Universe
Layer 3  Product     Wish Gate → Star Birth → My Star → Daily Loop
Layer 4  Platform    Galaxy Map → Star Share → Community Stars
Layer 5  Expansion   애니메이션 · 웹툰 · IP · 여행 · 이벤트
```

---

## Layer 1 — Origin

```
여수 바다
    ↓
Golden Turtle Constellation
    ↓
DreamTown 하늘
```

> 여수 바다에서 시작된 하늘.

---

## Layer 2 — Universe

```
소원 → 빛구슬 → 디지털 용궁 → 별씨 → StarLink → 별
                                              ↓
                                    별자리 → 은하 → DreamTown Universe
```

핵심 캐릭터: **아우룸** (황금 거북 안내자)

핵심 상징: 별 · Golden Turtle · 여수 바다

---

## Layer 3 — Product (현재 단계)

```
Wish Gate → Star Birth → My Star → Daily Message
```

| 요소 | 정의 |
|------|------|
| Core Action | 내 별 보기 |
| Core Loop | 소원 → 별 → 내 별 → 다시 보기 |
| Key Moment | 내 별을 다시 바라보는 순간 |
| Core Emotion | 조용한 위로 |

---

## Layer 4 — Platform

```
별 → 별자리 → 은하
```

주요 기능: Galaxy Map · Star Growth · Star Share · Community Stars

---

## Layer 5 — Expansion

DreamTown Universe:
```
위로 서비스 + 세계관 IP + 여행 경험 + 플랫폼
```

---

## 핵심 경험 (Star Birth)

```
소원 입력 → 빛 등장 → 아우룸 등장(무언) → 별 탄생 → 내 별
```

