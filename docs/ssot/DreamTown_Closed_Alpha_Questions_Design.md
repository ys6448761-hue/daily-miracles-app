# DreamTown Closed Alpha 질문 3개

**확정일**: 2026-03-12 (루미)
**원칙**: Alpha는 3개가 최선. 많으면 편향 발생.

---

## 질문 3개

### Q1. Star Birth 감정
> "별이 만들어지는 순간 어떤 느낌이었나요?"

**확인하는 것**: DreamTown 핵심 순간이 사람에게 감정을 주는가.

성공 반응:
- "오…"
- "신기하다"
- "조금 감동이다"

### Q2. 재방문 의사
> "이 앱을 다시 들어올 것 같나요?"

**확인하는 것**: 단순 재미 vs 습관 가능성.

DreamTown이 성공하려면 "한 번 하고 끝"이 아니라 "가끔 다시 들어오는 앱"이어야 한다.

### Q3. 가장 좋았던 순간
> "가장 마음에 들었던 순간은 언제였나요?"

**확인하는 것**: 사용자가 말하는 순간 = 제품의 진짜 핵심.

선택지 예시 (자유 응답 권장):
- Wish 작성
- 보석 선택
- Star Birth 애니메이션
- My Star 화면

---

## Alpha 진행 방식

1. 링크 보내기
2. 아무 설명 없이 체험
3. 다음 날 질문 3개

**절대 하면 안 되는 것**: 기능 설명, 세계관 설명, DreamTown 철학 설명.
사용자가 스스로 느끼게 해야 한다.
