## 1. Galaxy Mode 개념

DreamTown은 인간의 삶의 여정을 **우주 탐험**으로 표현한다.

```
Star (개인의 소원)
      ↓
Constellation (비슷한 여정들의 모임)
      ↓
Galaxy (테마별 컬렉션)
      ↓
Galaxy Cluster (대형 지식 도메인)
```

모든 여행자(소원이)는 소원 하나로 시작한다.
그 소원은 별(Star)이 되어 우주를 이룬다.

## 2. Origin Galaxy — 여수 은하 (Yeosu Galaxy)

### 출발점

DreamTown의 첫 번째 은하는 **여수 은하**다.

> **"DreamTown의 하늘은 여수의 바다에서 태어났다."**

### 여수가 출발점인 이유

| 요소 | 의미 |
|------|------|
| 디지털 용궁 (Dragon Palace) | 소원이 빛 구슬로 보관되는 바닷속 공간 |
| 아우룸 (황금 거북) | 바다와 별 사이를 잇는 안내자 |
| 바다 → 별 변환 | 소원이 현실에서 우주로 이어지는 상징 |

여수는 소원항해(Dream Voyage)의 **원항(Origin Port)**이다.

---

## 3. 여수 은하 구조

```
Yeosu Galaxy (여수 은하)
      ↓
Yeosu Voyage Route (여수 항해 항로)
      ↓
Courses (코스)
```

### 여수 항해 항로 코스 (4종)

| 코스 | 장소 | 핵심 감정 |
|------|------|----------|
| 향일암 일출 코스 (Hyangiram Sunrise) | 향일암 | 새로운 시작, 갱신 |
| 오동도 숨결 코스 (Odongdo Breath) | 오동도 해안길 | 설렘, 고요함 |
| 케이블카 시야 코스 (Cable Car Perspective) | 해상케이블카 | 이동, 여정, 조망 |
| 돌산 별빛 코스 (Dolsan Starlight) | 돌산공원 야경 | 여운, 다짐 |

## 4. 코스 정의

코스는 단순한 여행지가 아니다. **변화의 순간**을 담은 경험이다.

```
Location (장소)
      ↓
Emotion (감정)
      ↓
Action (행동)
      ↓
Miracle Moment (기적의 순간)
```

### 코스 예시 — 향일암 일출 코스

| 항목 | 내용 |
|------|------|
| **장소** | 향일암 |
| **감정** | 갱신 (Renewal) |
| **권유 행동** | 소원을 다시 써보세요 |
| **기적의 순간** | 결심 / 재다짐 (Decision / Recommitment) |

### 코스 예시 — 케이블카 시야 코스

| 항목 | 내용 |
|------|------|
| **장소** | 해상케이블카 |
| **감정** | 이동, 여정 |
| **권유 행동** | 지금 어디쯤 와 있는지 바라보세요 |
| **기적의 순간** | 관점의 전환 |

## 5. 아우룸 인터랙션

아우룸은 핵심 순간에 **조용히** 등장한다.
주도하지 않는다. 잠깐 나타났다가 사라진다.

### 등장 시점 및 메시지

| 시점 | 메시지 |
|------|--------|
| **코스 시작 전** | "어쩌면 여기서 작은 시작이 기다리고 있을지도 모릅니다." |
| **성찰 후** | "오늘 당신의 별이 조금 자랐을지도 모릅니다." |

### 아우룸 등장 원칙
- 경험을 지배하지 않는다
- 짧게 나타났다가 사라진다
- 말보다 존재로 신뢰를 전달한다

> 아우룸 외형 스펙: `DreamTown_Character_SSOT.md` 참조

## 6. 기적 카드 (Miracle Card)

은하 탐험 중 소원이는 **기적 카드**를 기록할 수 있다.

### 카드 구조

| 항목 | 선택지 |
|------|--------|
| **출처** | 일상 / 은하 항로 |
| **유형** | 행동 / 깨달음 / 연결 / 변화 |

### 유형별 예시

| 유형 | 예시 |
|------|------|
| 행동 (Action) | "향일암에서 소원을 다시 썼다." |
| 깨달음 (Realization) | "내가 원하는 게 뭔지 처음으로 분명해졌다." |
| 연결 (Connection) | "오동도에서 오래된 친구를 떠올렸다." |
| 변화 (Transformation) | "케이블카 위에서 두려움이 사라졌다." |

## 8. 여수 항로 — 여행 상품 연동

Galaxy Mode의 코스는 실제 여행 상품과 연결된다.

### 항로 체계 (4종)

| 항로 | 요일 | 포지션 |
|------|------|--------|
| **주중 항로** | 월~목, 일 | 가성비, 한적함, 커플·소그룹 |
| **별빛 항로** | 금·토·공휴일 | 프리미엄, 불꽃 감성, 기념일 |
| **소망 항로** | 전체 | Daily Miracles 고유, 소원 실현 체험 |
| **패밀리 항로** | 전체 | 3~4인 가족 |

### 코스 ↔ 여행 상품 매핑

| 코스 | 연결 상품 | 배경 코드 |
|------|----------|----------|
| 향일암 일출 | 소망 항로 (소원항해단) | `YS05` |
| 오동도 숨결 | 주중 항로 | `YS01` |
| 케이블카 시야 | 주중·별빛 항로 | `YS02` |
| 돌산 별빛 | 별빛 항로 | `YS03` |

> 상품 상세 가격: `DreamTown_Product_Structure_SSOT.md` 참조

## 9. 견적 시스템 연동

- **엔진**: `services/quoteEngine.js`
- **가격 데이터**: `config/quotePriceData.js` v1.2
- **요일 구분**: sun / mon-thu / fri / sat / holiday (5단계)
- **견적 유효기간**: 7일
- **견적 ID 형식**: `SW-YYYYMMDD-NNN`
- **리드 분류**: HOT(60점+) 즉시 연락 / WARM(40점+) 10분 내
