| 층 | 이름 | 역할 |
|----|------|------|
| 상층 | DreamTown Sky | 별 / 별자리 / 지혜 — 표현과 연결 |

## 핵심 흐름 (소원 → 지혜)

```
소원 (현실)
    ↓
빛구슬 / 종이비행기 (형태화)
    ↓
디지털 용궁 (보관)
    ↓
별씨앗 (숙성)
    ↓
Aurora Path / StarLink (변환)
    ↓
DreamTown 별 (탄생)
    ↓
Constellation (연결)
    ↓
Wisdom (지혜)
```

이 구조 하나로 제품 구조 + 세계관 + 데이터 흐름이 모두 설명된다.

## DreamTown 하늘 UX 구조

DreamTown은 SNS 피드가 아니라 **별 지도 + 탐험형 커뮤니티**다.

```
밤하늘 진입
    ↓
별 발견
    ↓
별 클릭
    ↓
이야기 탐험
    ↓
별자리 발견
```

## Constellation 영역 예시

### 창업 항해
```
준비 → 실패 → 재도전 → 성장
```

### 관계 회복
```
이별 → 자기 이해 → 화해 → 새로운 시작
```

Human Wisdom Network
