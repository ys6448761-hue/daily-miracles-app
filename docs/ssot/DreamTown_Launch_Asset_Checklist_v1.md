## 🟡 P1 — Soft Launch 전 필수

### 텍스트 자산
- [ ] 1문장 소개 확정
  > "DreamTown은 소원을 별로 만드는 조용한 위로 앱입니다."
- [ ] 짧은 소개글 (100자 이내)
- [ ] 핵심 메시지 3개 확정 (Launch Strategy 기준)
- [ ] CTA 문장 확정

### 비주얼 자산
- [ ] 대표 화면 스크린샷 3장 (Wish Gate / Star Birth / My Star)
- [ ] Star Card 대표 샘플 1종
- [ ] DreamTown 로고 파일 (PNG, SVG)
- [ ] 대표 컬러 팔레트 확인 (퍼플·핑크 그라디언트)

### 채널 자산
- [ ] 인스타그램 계정 설정
- [ ] 카카오 채널 (@dailymiracles) 소개 업데이트
- [ ] 앱 URL / 링크 준비 (app.dailymiracles.kr 또는 dreamtown 서브도메인)

---

## 🟢 P2 — Public Launch 전 필수

### 제품
- [ ] Star Share 기능 (이미지 저장 + 카카오 공유)
- [ ] Today Message 기능
- [ ] Galaxy Map Lite
- [ ] 공개 별 URL (`/star/:id`)

### 마케팅 자산
- [ ] 10초 데모 영상 (소원 입력 → 별 탄생 → 내 별 보기)
- [ ] 인스타그램 릴스 1편
- [ ] Star Card 5종 이상
- [ ] 카카오 공유 템플릿

### 운영 준비
- [ ] 오류 모니터링 설정
- [ ] 기본 사용 지표 수집 (별 생성 수 / My Star 재방문)
- [ ] 피드백 수집 채널 (오픈채팅 또는 폼)
