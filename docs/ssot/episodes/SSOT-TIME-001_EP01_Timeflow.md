# SSOT-TIME-001 — EP01 별빛항로 타임플로우

## 목적

EP01 별빛항로의 실제 운영 시간 기준을 고정한다.

여름철 유람선 시간, 케이블카, 하멜등대 이후 자유 선택 구조를 반영한 운영 기준이다.

> 이 문서는 제작 기준이 아닌 **실제 운영 기준**이다.
> 가격·일정 변경 시 이 문서를 먼저 업데이트한다.

## 23:30 이후 — 자유 선택

DreamTown은 패키지 강제 일정이 아니다.

하멜등대 이후 일정은 고객 선택에 맡긴다. 24시 이후까지 공식 일정을 강제하지 않는다.

**선택 예시:**
- 유탑호텔 이동
- 낭만포차 이용
- 야간 산책
- 자유 휴식

**운영 문구:**

> 별빛 항로는 하멜등대에서 끝나지만,
> 그 이후의 밤은 당신만의 여수로 이어집니다.
