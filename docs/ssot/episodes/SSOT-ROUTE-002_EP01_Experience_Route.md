# SSOT-ROUTE-002 — EP01 별빛항로 실제 체험 Route

## 목적

실제 고객이 체험하는 1박 2일 별빛항로의 운영 기준을 고정한다.

> **영상 Route(SSOT-ROUTE-001)와 이 문서는 목적이 다르다.**
>
> | 구분 | Route | 종료 지점 |
> |------|-------|----------|
> | 영상 콘텐츠 | SSOT-ROUTE-001 | 하멜등대 |
> | 실제 체험 | SSOT-ROUTE-002 (이 문서) | Day 2 귀가 |

DreamTown의 실제 항로는
소원이 빛나기 시작한 마음을 가지고 **일상으로 돌아갈 때 완성된다.**

---

## Day 1 — 감정 항로

```
도착 → 호흡 → 연결 → 상승 → 쉼 → 소원 → 안식
```

### Day 1 시간표

| 시간 | 장소 | 감정 역할 |
|------|------|----------|
| 10:00 ~ 11:00 | 여수엑스포역 | 도착 / 낯섦 / 기대 |
| 11:00 ~ 13:00 | 엑스포 바닷길 | 호흡 / 긴장 완화 |
| 13:00 ~ 14:00 | 점심 (자유) | 여백 |
| 14:00 ~ 16:00 | 이순신광장 | 연결 / 도시의 온기 |
| 16:00 ~ 17:30 | 케이블카 | 상승 / 시야 확장 |
| 17:30 ~ 19:30 | 저녁 (자유) | 여백 |
| 20:20 ~ 21:50 | 유람선 (미남크루즈) | 상승의 확장 / 야경 |
| 22:00 ~ 23:00 | 종포해양공원 | 쉼 / 정리 / 고요 |
| 23:00 ~ 23:30 | 하멜등대 | **소원 ★ EP01 감정 클라이맥스** |
| 23:30 ~ | 자유 시간 | 당신만의 여수 |

---

## Day 2 — 감정 항로

```
재충전 → 사색 → 발견 → 현실 연결 → 귀환
```

### Day 2 시간표

| 시간 | 장소 | 감정 역할 |
|------|------|----------|
| 09:00 ~ 10:30 | 오동도 | 재충전 / 자연 회복 |
| 10:40 ~ 12:00 | 모이핀 | 사색 / 감정 정리 |
| 12:00 ~ 13:00 | 점심 (자유) | 여백 |
| 13:00 ~ 15:00 | 장도 | 발견 / 소원 재확인 |
| 15:00 ~ 16:00 | 수산시장 · 건어물시장 | 현실 연결 / 여수를 가지고 간다 |
| 17:00 ~ 18:00 | 귀가 | **귀환 ★ 체험 Route 완성** |

---

## Day 2 최종 감정

> **집으로 돌아가도 괜찮을 것 같다.**

---

## 체험 Route 설계 원칙

1. Day 1 클라이맥스(하멜등대) 이후 Day 2가 이어진다
2. Day 2는 소원을 가지고 현실로 돌아가는 과정이다
3. Day 2 종료(귀가)가 체험 Route의 완성이다
4. 강제 일정 없음 — 각 장소는 권장이며 고객 상황에 따라 조정 가능
5. 운영 편의보다 감정 흐름을 우선한다
