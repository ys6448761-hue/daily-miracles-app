# SSOT-ROUTE-001 — EP01 소원 여정 항로

## 목적

같은 장소라도 순서가 바뀌면 감정이 완전히 달라진다.

항로(Route)는 장소의 **감정 흐름 순서**를 고정한다.

> 여수역 → 하멜등대 는 소원 여정이 된다.
> 하멜등대 → 여수역 은 소원 여정이 될 수 없다.

---

## EP01 항로 전체 구조

```
[1] 여수엑스포역       — 시작 / 도착
        ↓
[2] 엑스포 바닷길      — 전환 / 호흡
        ↓
[3] 이순신광장         — 연결 / 사람의 온기
        ↓
[4] 케이블카           — 상승 / 희망 확장
        ↓
[5] 종포해양공원       — 쉼 / 정리
        ↓
[6] 하멜등대           — 소원 / 별
        ↓
[7] 호텔               — 안식 / 내려놓음
```

## 항로 설계 원칙

1. 항로는 반드시 **도착 → 전환 → 연결 → 상승 → 쉼 → 소원 → 안식** 순서를 따른다
2. 장면을 생략할 수 있으나 순서를 역행할 수 없다
3. 하멜등대는 항상 클라이맥스 위치(마지막에서 두 번째)를 유지한다
4. 호텔(안식)은 항상 마지막 장면이다

---

## 역순 설계 기준

EP01은 시작점이 아니라 **엔딩 감정**에서 설계됐다.

> EP01 최종 감정:
> "여수에서, 내 소원이 다시 빛나기 시작했다."

이 감정에 도달하기 위해 항로는 역산으로 배치됐다.

```
안식 (엔딩)
  ← 소원
  ← 쉼
  ← 상승
  ← 연결
  ← 호흡
  ← 도착 (시작)
```

정방향 실행 순서는 위의 역순이다.
다음 에피소드도 반드시 **엔딩 감정 문장을 먼저 정의**한 뒤 항로를 역산한다.
