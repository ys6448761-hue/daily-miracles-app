## 📌 설계 목적

DreamTown은 현재 텍스트 위주 화면으로 구성되어 있다.
소원이의 동기 부여와 성장 체감을 위해
My Star 화면에 시각화 요소를 단계적으로 도입한다.

핵심 원칙:
- 숫자보다 감성
- 경쟁보다 성장
- 데이터보다 서사

## 🏗️ My Star 화면 구조 (시각화 포함)

### 1. 헤더 — 별 정체성

```
✦ 꽃을 스미는 별
성장 은하 · 2026.03.23 탄생
```

### 2. 요약 카드 3개 (가로 배치)

| 카드 | 표시 내용 | 보조 설명 |
|------|----------|----------|
| 항해 일수 | D+N | D+30까지 N일 |
| 변화 지수 | 0~100 | 지난 주보다 ±N |
| 공명 받음 | N회 | N명이 응원해요 |

### 3. 항해 여정 — 마일스톤 진행바

```
D+1 ──✓── D+7 ──✓── 지금(●) ────── D+30 ────── D+100 ────── D+365
```

- 완료 마일스톤: 골드 채움 (✓)
- 현재 위치: 강조 점 (●)
- 미래 마일스톤: 흐린 원 (숫자)

### 4. 변화 지수 그래프

```
Chart.js 라인 차트
X축: D+1 ~ 현재 (최대 D+N)
Y축: change_score (0~100)
색상: Star Gold (#FFD76A)
배경: 골드 그라데이션 페이드
강조: 현재 포인트 크게 표시
```

데이터 소스: `dt_voyage_logs` change_score 누적

#### Chart.js 스펙
```
- borderColor: #FFD76A
- borderWidth: 2
- fill: true (그라데이션)
- tension: 0.4 (부드러운 곡선)
- pointRadius: 현재 포인트만 5, 나머지 2
- 배경: Night Sky 계열
- 그리드: rgba(255,255,255,0.05)
```
