## 핵심 축 (Origin Axis)

DreamTown 우주 구조의 중심선:

```
현실 세계
    ↓
여수 바다
    ↓
Golden Turtle Constellation (황금 거북 별자리)
    ↓
DreamTown 은하
```

> **"여수 바다에서 시작된 하늘."**

---

## Layer 구조

### Layer 1 — Ocean Origin (바다)

소원의 탄생 장소. DreamTown 세계의 최하층.

```
소원
    ↓
빛구슬
    ↓
별씨앗
```

### Layer 2 — Golden Turtle Field (황금 거북 영역)

여수 실제 지형과 연결된 세계관 기원 영역.

| 실제 지형 | 세계관 역할 |
|----------|------------|
| 백도 | |
| 거문도 | |
| 사도 | |
| 금오도 | |
| 향일암 | |
| 오동도 | |
| 제석산 | |
| 신풍 | |

이 지역들이 **Golden Turtle Field** — Golden Nine Constellation의 실제 기원지.

### Layer 3 — Star Birth Gate (별 탄생 관문)

```
빛구슬
    ↓
아우룸 등장
    ↓
StarLink (Aurora Path)
    ↓
별 탄생
```

→ **Star Birth Scene** (DreamTown 핵심 장면)

### Layer 4 — DreamTown Sky (하늘)

소원들이 별이 되어 존재하는 세계.

별 구성 요소:
- 별 이름
- 소원이 닉네임
- 소원 내용
- 항해 기록
- 깨달음
- 나눔 반응

### Layer 5 — Constellations (별자리)

비슷한 여정을 가진 별들이 모여 별자리를 이룬다.

별자리 예시:
- 도전 별자리
- 치유 별자리
- 관계 별자리
- 성장 별자리

별자리 = **인간 경험의 패턴**

### Layer 6 — Galaxies (은하)

별자리가 모여 은하를 이룬다.

| 은하 | 의미 |
|------|------|
| 도전 은하 | 새로운 시작 |
| 성장 은하 | 자기 발견 |
| 관계 은하 | 사랑과 연결 |
| 치유 은하 | 마음 회복 |

## DreamTown Compass (방향 지도)

```
          북쪽 — 도전
             ↑
서쪽 — 관계  ←  →  동쪽 — 성장
             ↓
          남쪽 — 치유
```

이 방향 구조가 **앱 탐험 지도**의 기준이 된다.

---

## Galaxy Expansion (은하 확장 로드맵)

```
여수 은하 (Origin)  ← 현재
    ↓
한국 은하
    ↓
세계 은하
```

사용자가 늘어나면 새로운 은하가 자연 생성된다.

| 은하 | 상태 |
|------|------|
| 여수 은하 | 운영 중 |
| 서울 은하 | 계획 |
| 부산 은하 | 계획 |
| 제주 은하 | 계획 |
| 교토·발리·산토리니 | 장기 |

## 소원이의 항해 여정

```
소원이 (소원 남김)
    ↓
항해 (여정 시작)
    ↓
깨달음 (성장)
    ↓
소망이 (다른 이의 길을 밝히는 존재)
```

## DreamTown의 목적

> **인간 경험의 은하 지도**

사람들의 소원과 경험이 별과 별자리로 기록되고,
그 기록이 다른 사람의 항해를 밝힌다.

---

## Cosmic UX (앱 구조 연결)

이 우주 지도가 그대로 앱 UX 흐름이 된다:

```
소원 입력
    ↓
별 탄생
    ↓
첫 항해
    ↓
별자리 발견
    ↓
은하 탐험
    ↓
다른 별 이야기 발견
```

DreamTown = **우주 탐험형 커뮤니티 플랫폼**

---

## 전략 포지셔닝

```
현실 지형 (여수)
+
우주 세계관 (DreamTown)
+
커뮤니티 플랫폼 (소원이 공동체)
```

현실 지형 기반 우주 세계관 — 세계적으로 드문 구조.
