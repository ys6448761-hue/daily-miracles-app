## 세계관 선언

> **여수 바다에서 시작된 하늘.**

> **당신의 소원은 혼자가 아닙니다.**

DreamTown은 인간의 소원이 바다에서 태어나 별이 되는 세계이다.

---

## 1. 세계 전체 구조

```
현실 세계 (여수 바다)
        ↓
    빛구슬 (소원)
        ↓
  디지털 용궁 (보관)
        ↓
    별씨 (Star Seed)
        ↓
    StarLink (변환)
        ↓
  DreamTown 하늘 (별)
        ↓
  별자리 (Constellation)
        ↓
    은하 (Galaxy)
        ↓
DreamTown Universe
```

사용자의 소원 하나가 이 여정 전체를 거친다.

---

## 2. 세계 구성 요소

### Origin — 여수 바다

| 항목 | 내용 |
|------|------|
| 위치 | 대한민국 여수 |
| 역할 | 소원의 시작점 / DreamTown 우주의 원점 |
| 의미 | 모든 이야기가 시작되는 바다 |

### 디지털 용궁 (Dragon Palace)

| 항목 | 내용 |
|------|------|
| 역할 | 소원이 잠드는 성소 |
| 기능 | 소원 보관 / 별씨 성장 / StarLink 준비 |
| 의미 | 단순한 DB가 아닌 소원의 안식처 |

### StarLink

| 항목 | 내용 |
|------|------|
| 역할 | 소원 → 별로 변환하는 과정 |
| 감정 | DreamTown 핵심 장면 (Star Birth) |

### Aurum (아우룸)

| 항목 | 내용 |
|------|------|
| 정체 | 황금 거북 안내자 |
| 역할 | 소원 수신 / 별 탄생 목격 / 은하 안내 |
| 원칙 | 주인공이 아닌 조용한 안내자 |

---

## 3. 별 (Star) — 세계 기본 단위

```
별 이름 / 소원 원문 / 사용자 / 탄생일 / 은하 / 성장 단계(Day N)
```

별은 사용자의 소원이 변환된 존재이며 DreamTown 세계의 기본 단위이다.

---

## 4. 은하 나침반 (Galaxy Compass)

```
           North
    Challenge Galaxy    (도전·용기)
           ⬆
West              East
Relationship  ←  ·  →  Growth
(관계·사랑)             (성장·변화)
           ⬇
    Healing Galaxy      (치유·회복)
           South
```

각 은하는 인간 삶의 중요한 영역을 상징한다.

## 8. DreamTown 핵심 정의

### 제품 정의
```
DreamTown은 소원을 별로 보내고 다시 그 별을 보러 오는 루틴형 위로 서비스이다.
```

### 세계관 정의
```
DreamTown은 사람들의 소원이 별이 되어 우주를 이루는 세계이다.
```

### 한 문장 정의
```
여수 바다에서 시작된 하늘, 그리고 별이 된 소원들의 도시.
```

---

## 9. DreamTown 정체성

```
세계관 + 제품 + 위로 서비스
```

| 관점 | 정의 |
|------|------|
| 세계관 | 소원이 별이 되는 우주 |
| 제품 | 루틴형 위로 앱 |
| 서비스 | 소원 보관 + 별 탄생 + 재방문 경험 |

이 세 가지 결합은 드문 모델이다.
