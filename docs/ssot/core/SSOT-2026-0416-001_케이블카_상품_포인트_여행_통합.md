# SSOT-2026-0416-001 — DreamTown 케이블카·포인트·여행 통합 구조

**버전**: 1.0  
**확정일**: 2026-04-16  
**상태**: 확정 (변경 금지)

---

## 1. 핵심 루프 (최상위 헌법)

```
감정(별) → 기록 → 포인트 → 여행 → 재방문
```

DreamTown은 예약 서비스가 아니라 **재방문 시스템**이다.

---

## 2. 케이블카 경험 구조 (무료 vs 유료)

### 무료
- 감정 기록 가능
- 장소 로그 가능
- **별 각성 연출 없음**

### 유료 상품 (별 각성 상품)
- **별 각성 연출 제공** (시그니처: 별이 점점 커지며 나에게 다가오는 장면)
- 감정이 폭발하는 시그니처 순간 제공
- 내 별/기억 확장과 연결

> **핵심 한 줄**: 케이블카는 장소이고, 각성은 상품이다.

---

## 3. 케이블카 상품 구조

### A. 베이직 (19,900원)
- 역할: 지금 이 순간을 기록하고 내 별에서 다시 볼 수 있게 만드는 입문형
- 핵심 가치: 기록 / 저장 / 내 별 연결

### B. 프리미엄 (24,900원)
- 역할: 하루의 기억을 더 넓게 연결하고 카페/여행지 확장 기록을 남기는 상품
- 핵심 가치: 기록 확대 / 기억 확장 / 장소 확장

---

## 4. 여수 미션 원칙

- QR 부착 필수 아님 (공공장소 특성)
- 사용자가 "지금 있는 장소"를 선택 → 감정 남기기
- 미션 감정 → 포인트 → 상품 자격/혜택/경험 unlock과 연결

---

## 5. 여행상품 2트랙 구조

### A. Basic
- 포인트 없이 구매 가능
- 일반형 상품
- 역할: 유입

### B. DreamTown
- 5,000P 이상 보유 시 접근 가능
- 경험 레벨이 다른 구조 (단순 할인판 아님)
- 역할: 충성 사용자 전용 경험

---

## 6. 상품 자격 조건 요약

| 상품 유형 | 포인트 조건 | 설명 |
|-----------|------------|------|

| 여행 Basic | 없음 | 누구나 |
| 여행 DreamTown | 5,000P 이상 | 충성 사용자 전용 |
