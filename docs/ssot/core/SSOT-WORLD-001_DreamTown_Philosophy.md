# SSOT-WORLD-001 — DreamTown 철학 선언

> 기존 Core Philosophy SSOT의 상위 제작 철학 레이어.
> Character / Visual / Route SSOT를 대체하지 않는다. 이 문서는 모든 제작 결정의 출발점이다.

---

## DreamTown은 무엇인가

DreamTown은 관광 홍보 프로젝트가 아니다.

DreamTown은 **감정 회복 설계(Emotional Recovery Design) 프로젝트**다.

---

## DreamTown의 목적

DreamTown의 목적은 사람들에게 답을 주는 것이 아니다.

> 사람들이 자신의 답을 **다시 찾을 수 있도록 돕는 것**이다.

소원은 이루어지는 것이 아니라,

> **다시 빛나기 시작하는 것이다.**

---

## 핵심 원칙 (5가지)

| 원칙 | 의미 |
|------|------|
| 장소는 캐릭터다 | 여수의 장소는 배경이 아니라 감정 역할을 수행하는 존재다 |
| 순서는 스토리다 | 같은 장소도 순서가 바뀌면 감정이 달라진다 |
| 회복은 목적지가 아니라 항로다 | DreamTown은 회복의 결과가 아닌 과정을 설계한다 |
| DreamTown은 희망을 강요하지 않는다 | 슬픔을 지우는 것이 아니라, 그 위에 희망이 올라오도록 돕는다 |
| DreamTown은 자기 안의 희망을 다시 발견하게 돕는다 | 외부에서 희망을 주입하지 않는다 |

---

## DreamTown 핵심 감정 흐름

```
소원 → 별 → 성장 → 공명 → 나눔 → 연결
```

이 흐름은 모든 에피소드, 캐릭터, 항로 설계의 상위 구조다.
새로운 SSOT나 에피소드는 이 흐름과 충돌할 수 없다.

---

## 제작 판단 기준

모든 제작 결정은 아래 3가지 질문을 통과해야 한다.

1. DreamTown답나?
2. 감정이 먼저 오나?
3. 회복을 강요하지 않나?

---

## 금지 연출 원칙

DreamTown은 다음을 만들지 않는다.

- 관광 홍보 콘텐츠
- 희망을 강요하는 연출
- 슬픔을 부정하는 내러티브
- 주인공이 사라지는 엔딩
- 도달해야 할 목적지로서의 회복
