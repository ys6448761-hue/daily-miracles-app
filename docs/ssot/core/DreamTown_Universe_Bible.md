## 1. 세계관 한 줄 정의

> **"여수 바다에서 시작된 하늘 — 당신의 소원은 혼자가 아닙니다."**

DreamTown은 인간의 소원이 별이 되어 서로를 비추는 디지털 우주 세계관입니다.
여수 앞바다의 전설(금오설화)에서 시작해, 소원이 빛구슬이 되고 하늘의 별이 되는 여정을 담고 있습니다.

### 공식 통합 정의 문장

> 모든 소원은 먼저 디지털 용궁에 닿습니다.
> 온라인의 소원은 빛구슬이 되어, 오프라인의 소원은 종이비행기가 되어 용궁으로 향합니다.
> 용궁에 보관된 소원은 아우룸과 Aurora5의 해석을 거쳐 별씨앗이 되고,
> StarLink를 통해 DreamTown의 별로 떠오릅니다.

## 2. Origin — 여수 바다와 아우룸

### 2-1. 기원 신화 (여수 금오설화 기반 창작 재해석)

> 본 신화는 여수 금오설화를 바탕으로 한 창작 재해석입니다.
> 여수시의 공식 입장을 대변하지 않습니다.

아주 오래전, DreamTown의 하늘이 존재하기 전
여수 앞바다 깊은 곳에는 신비로운 **용궁**이 있었습니다.

용궁에는 황금빛 등껍질을 가진 거북들이 살고 있었습니다.
그들을 **금오(金鼇)** — 황금 거북이라 불렀습니다.
그중 아홉 형제가 있었습니다.

용왕은 아홉 형제에게 임무를 주었습니다.

> "인간 세상을 살펴보고 돌아오너라."

형제들은 여수 앞바다로 올라왔습니다.
푸른 바다, 섬과 별빛, 인간들의 작은 삶.
형제들은 그 아름다움에 마음을 빼앗겨 복귀 시간을 놓쳐버렸습니다.

용왕의 벌로 아홉 형제는 여수 바다의 **수호 바위**가 되었습니다.
그래서 지금도 여수 곳곳에 거북 형상의 바위와 지형이 남아 있습니다.

### 2-2. DreamTown 창작 설정 — 금오 9형제

| 설정 | 내용 |
|------|------|
| 1~8 형제 | 여수 지형에 잠든 수호 별 (고정된 별자리) |
| 9번째 아우룸 | 유일하게 자유롭게 움직이는 황금 거북 |

형들이 바위가 된 이후, 아우룸은 깨달았습니다.
인간들은 바다를 향해 조용히 소원을 빌고 있었습니다.

> 사랑을 위해. 가족을 위해. 꿈을 위해.

아우룸은 결심했습니다.

> **인간들의 소원을 용왕님께 전달하기로.**

### 2-3. 아우룸 (Aurum) — 공식 정의

> **아우룸은 디지털 용궁의 황금 거북 수호자이자,
> 인간의 소원을 바다와 하늘 사이로 인도하는 DreamTown의 내비게이터입니다.
> 온라인의 빛구슬과 오프라인의 종이비행기를 용궁으로 전달하고,
> 잠든 소원이 별씨앗에서 별로 떠오를 때까지 조용히 지켜봅니다.**

#### 아우룸의 3가지 역할

| 역할 | 설명 |
|------|------|
| 소원 전달자 | 인간의 소원을 용궁으로 전달 |
| 용궁 수호자 | 소원을 안전하게 보관·지키는 존재 |
| 별 안내자 | 별씨앗이 별이 되는 순간 하늘로 안내 |

#### 아우룸 외형 스펙 (AURUM TURTLE LOCK v2)

| 얼굴 | 두 개의 작은 눈, 작은 부리형 입, 두 개의 콧구멍 점 |

> **아우룸은 주인공이 아닙니다. 주인공은 언제나 소원이의 소원입니다.**

## 3. 5개 은하 구조

소원은 별이 되고, 별들이 모여 은하를 이룹니다.
DreamTown은 5개의 은하로 이루어져 있습니다.

### 3-1. DreamTown Compass — 5은하 배치

```
              North — 도전 (Challenge)
                    ↑
West — 관계    ←   ✦   →   East — 성장
                    ↓
              South — 치유 (Healing)

              Center — 기적 (Miracle) ✨
```

### 3-2. 5개 은하 상세 정의

#### ① 도전 은하 (Challenge Galaxy)
- **방향**: 북쪽 (North)
- **보석**: 루비 🔴
- **의미**: 용기를 내어 앞으로 나아가려는 마음
- **소원 유형**: 두려움 극복, 새로운 시작, 결단
- **상징**: 뛰어오르는 불꽃

#### ② 성장 은하 (Growth Galaxy)
- **방향**: 동쪽 (East)
- **보석**: 사파이어 🔵
- **의미**: 더 나은 나를 향해 배우고 싶은 마음
- **소원 유형**: 자기계발, 학습, 커리어, 능력 향상
- **상징**: 새벽빛

#### ③ 치유 은하 (Healing Galaxy)
- **방향**: 남쪽 (South)
- **보석**: 에메랄드 🟢
- **의미**: 마음의 상처를 보듬고 쉬어가고 싶은 마음
- **소원 유형**: 회복, 안식, 자기수용, 마음 돌봄
- **상징**: 잔잔한 바다

#### ④ 관계 은하 (Relationship Galaxy)
- **방향**: 서쪽 (West)
- **보석**: 시트린 🟡
- **의미**: 소중한 관계를 더 깊게 이어가고 싶은 마음
- **소원 유형**: 사랑, 우정, 가족, 연결
- **상징**: 이어진 별빛

#### ⑤ 기적 은하 (Miracle Galaxy) ✨
- **방향**: 중심 (Center)
- **보석**: 다이아몬드 💎
- **의미**: 하나의 은하에 담기지 않는, 가장 순수하고 간절한 소원
- **소원 유형**: 삶 전체를 바꾸는 소원, 기적처럼 이루어지길 바라는 소원
- **상징**: 모든 빛이 모이는 중심점
- **특징**: 4개 은하의 중심에 위치. 극히 드물게 배정되는 특별한 은하

### 3-3. 4계층 우주 구조

```
Star (개인의 소원)
      ↓
Constellation (비슷한 여정의 모임)
      ↓
Galaxy (테마별 컬렉션 — 5개 은하)
      ↓
Galaxy Cluster (대형 지식 도메인)
```

## 4. 별빛 공방 (Byeolbit Gongbang — Starlight Workshop)

**별빛 공방**은 DreamTown 세계관 안에서 소원이 시각 예술로 변환되는 창작 공간입니다.

### 4-1. 정의

> **별빛 공방은 인간의 소원을 별의 언어로 번역하는 곳입니다.**
> 아우룸이 용궁으로 가져온 소원은 이 공방에서 빛구슬로, 소원그림으로, 성장필름으로 만들어집니다.

### 4-2. 별빛 공방의 3가지 산출물

| 산출물 | 설명 | 기술 기반 |
|--------|------|----------|
| **소원그림 (Wish Image)** | 소원을 AI가 시각화한 이미지 | DALL-E 3 |
| **성장필름 (Growth Film)** | 항해 완주 후 만들어지는 소원 여정 영상 | Aurora5 영상 파이프라인 |
| **별 탄생 씬 (Star Birth)** | 소원이 DreamTown에서 별이 되는 애니메이션 씬 | 2D 수채화 스타일 |

### 4-3. 별빛 공방 철학

```
소원이 말로만 존재할 때: 잊혀진다
소원이 그림이 될 때: 눈에 보이기 시작한다
소원이 별이 될 때: 하늘에 남는다
```

별빛 공방은 보이지 않는 소원에 형태를 부여하는 창작 시스템입니다.
모든 소원은 별빛 공방을 거쳐 시각적 실체를 얻습니다.

### 4-4. 별빛 공방 위치

- **세계관 내 위치**: 디지털 용궁 내부, 아우룸이 관리하는 창작 공간
- **현실 대응**: Aurora5 AI 시스템 + 기술 파이프라인

## 5. 소원이 → 소망이 성장 구조

DreamTown 사용자(소원이)는 여정을 통해 성장합니다.
이 성장은 단계적으로 진행되며, 최종 단계는 **소망이(Somanggi)**입니다.

### 5-1. 정체성 정의

| 이름 | 영문 | 의미 |
|------|------|------|
| **소원이 (Sowoni)** | Sowoni | 소원을 품고 여정을 시작한 사람 |
| **소망이 (Somanggi)** ✨ | Somanggi | 자신 안의 빛을 발견하고 다른 이들의 길을 밝히는 사람 |

> 소망이는 목적지가 아닌 **상태**입니다. 도달하는 것이 아니라, 되어가는 것입니다.

### 5-2. 별 성장 5단계

```
① Seed Star (씨앗별)
   소원을 처음 품은 상태 — 아직 빛이 약하다

        ↓

② Growing Star (성장별)
   항해를 시작하고, 기적 순간을 기록하는 중

        ↓

③ Bright Star (빛나는 별)
   항해를 완주하고, 소원이 선명해진 상태

        ↓

④ Guide Star (안내별)
   다른 소원이의 항해를 비추기 시작한 별

        ↓

⑤ Somangi ✨ (소망이)
   자신의 여정이 다른 이의 길을 밝히는 별
   "소원이"에서 "소망이"로의 완성
```

### 5-3. 항해 여정 구조

```
소원 (Wish)       ← 용궁에 보관
      ↓
항해 (Journey)    ← 소원이 → 소망이로 성장하는 여정
      ↓
깨달음 (Realization)  ← 여정에서 얻은 지혜
      ↓
나눔 (Sharing)    ← 지혜를 별자리로 나누는 행위
```

### 5-4. 소원이 외형 스펙 (CHARACTER LOCK v2)

| 속성 | 설정 |
|------|------|
| 역할 | 소원을 품은 모든 사람을 대표하는 메인 내러티브 캐릭터 |
| 연령 | 20~22세 한국 대학생 (성인 명시) |
| 스타일 | 웹툰 큰 눈, 단순한 코, 부드럽고 둥근 얼굴 |
| 라인 | 단순 애니 선화, 셀 쉐이딩 |
| 금지 | 교복, 아동 느낌 연출, 옷/소지품에 텍스트 |

## 6. 별 이름 시스템 (Star Naming System)

DreamTown의 모든 별은 고유한 이름을 가집니다.

### 6-1. 별 이름 생성 원칙 (B+C 방식)

```
B: 소원이 직접 입력한 핵심 단어
+
C: Aurora5(아우룸)가 소원에서 추출한 감정·방향성 단어
= 별 이름
```

예시:
```
소원: "매일 조금씩 글을 쓰고 싶어요"
→ 별 이름: "조용한 글의 항해"

소원: "엄마와 더 자주 통화하고 싶어요"
→ 별 이름: "온기의 연결"

소원: "다시 용기를 내고 싶어요"
→ 별 이름: "다시 시작하는 용기"
```

### 6-2. 별 이름 언어 원칙

| 원칙 | 내용 |
|------|------|
| **시적 감수성** | 평이한 문장이 아닌, 시처럼 울리는 이름 |
| **소원이의 언어** | 소원이가 쓴 단어를 존중 |
| **미래지향** | 현재 상태가 아닌 되어가는 방향 |
| **간결함** | 2~5단어 이내 |

### 6-3. 별 구성 요소

| 필드 | 내용 |
|------|------|
| Star Name | 별 이름 (B+C 방식) |
| Sowoni | 소원이 닉네임 |
| Galaxy | 소속 은하 (5개 중 1개) |
| Wish | 소원 내용 |
| Journey Logs | 항해 기록 |
| Resonance | 공명 반응 (공명자들의 마음) |

### 6-4. 공명 시스템 (Resonance)

소원이의 별에 다른 이들이 마음을 보내는 4가지 공명 유형:

| 유형 | 의미 |
|------|------|
| `relief` | 좀 편해졌어요 |
| `courage` | 좀 용기났어요 |
| `clarity` | 좀 정리됐어요 |
| `belief` | 좀 믿고 싶어졌어요 |

공명은 소원이의 별에 닿은 마음입니다.
공명자는 자신을 드러내지 않고, 조용히 마음을 보냅니다.

### 7-3. 신호등 케어 시스템

|------|------|-------------|
| 🔴 **RED** | 위기 상태 감지 (자살/자해 관련 키워드) | 즉시 관리자 알림 + 위기 대응 메시지 + 별 생성 중단 |
| 🟡 **YELLOW** | 주의 상태 (감정적 어려움 신호) | 따뜻한 응원 메시지 + 모니터링 강화 |
| 🟢 **GREEN** | 정상 여정 | 기적 기록 안내 + 조용한 동행 |

#### RED 신호 원칙

```
DreamTown은 치료 서비스가 아닙니다.

RED 감지 시, 별 생성을 멈추고 사람을 먼저 봅니다.
```
