## 핵심 정의

> **포인트는 현금이 아니라 '여수 재방문 자산'이다**

---

## 포인트 구조

| 항목 | 기준 |
|------|------|
| 환산율 | 1P = 내부 기준 10원 |
| 사용처 | 숙소 / 케이블카 / 이벤트 / 지역 소비 |
| 성격 | 재방문 유도 자산 (현금 전환 불가) |

---

## 제휴 구조

| 유형 | 역할 | 핵심 지표 |
|------|------|-----------|
| 케이블카 | 유입 엔진 | 신규 별 탄생 수 |
| 호텔 | 고향 엔진 | 고향 등록 수 / 재방문 횟수 |
| 카페·식당·레저 | 로그 엔진 | 로그 기록 수 / 체류 시간 |

---

## 수익 구조

- 객실 직접 판매 X
- 고객 유입 → 부가 소비 발생 → 파트너 매출 증가
- 재방문 → 반복 매출 → 지역 경제 순환

---

## 핵심 전략

1. **빈 객실을 고객 유입으로 전환** — 비수기 객실을 DreamTown 포인트 교환 상품으로 활용
2. **고객 → 소비 → 재방문 사이클** — 포인트 잔액이 재방문 동기 생성
3. **감정 연결 → 경제 연결** — 별/고향 감정 구조가 실제 소비로 이어짐
