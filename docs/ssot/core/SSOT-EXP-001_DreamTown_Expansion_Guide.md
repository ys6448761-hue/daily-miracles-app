# SSOT-EXP-001 — DreamTown 확장 가이드

## 목적

DreamTown은 항로 영상에서 시작했지만, 감정 회복 설계 프로젝트로서 확장 가능하다.

이 문서는 확장 방향과 수준을 정의하며, 각 레벨은 이전 레벨이 완성된 뒤 시작한다.

---

## 확장 레벨 구조

### Level 1. 감정 항로 (현재)

에피소드별 감정 항로를 제작한다.

| 에피소드 | 항로명 | 엔딩 감정 |
|---------|--------|----------|
| EP01 | 별빛 항로 | 소원이 다시 빛나기 시작 |
| EP02 | 달빛 항로 | 마음이 열리기 시작 |
| EP03 | 소망 항로 | 소원을 소리 내어 말함 |

---

### Level 2. 감정 지도

여러 항로가 모이면 **DreamTown 감정 지도**가 된다.

각 항로는 하나의 감정 경로를 나타내며, 지도는 여수 전체를 감정 레이어로 덮는다.

---

### Level 3. 감정 네비게이션

사용자의 현재 감정 상태에 따라 적합한 항로를 추천한다.

| 사용자 상태 | 추천 항로 |
|------------|----------|
| 지쳐 있다 | 호흡 → 쉼 → 안식 |
| 새로운 시작이 두렵다 | 도착 → 연결 → 상승 |
| 희망을 잃었다 | 쉼 → 소원 → 안식 |
| 혼자라는 느낌 | 연결 → 상승 → 소원 |

---

### Level 4. 현실 여행

DreamTown Route를 **실제 여수 여행 동선**으로 확장한다.

관광이 아닌 감정 회복을 위한 여행 동선으로 설계된다.

---

### Level 5. DreamTown Universe

소원이 외 다른 캐릭터와 다른 감정 항로로 확장 가능하다.

> 새로운 캐릭터는 소원이의 세계관과 충돌하지 않아야 한다.
> 새로운 항로는 SSOT-LOC-001 장소 감정 역할과 충돌하지 않아야 한다.

---

## 확장 원칙

1. 확장은 기존 SSOT를 **대체하지 않는다** — 레이어를 추가한다
2. 각 레벨은 이전 레벨이 안정화된 뒤 시작한다
3. 새 캐릭터·장소·항로는 SSOT-WORLD-001 철학과 충돌할 수 없다
4. 핵심 감정 흐름 `소원 → 별 → 성장 → 공명 → 나눔 → 연결` 은 모든 확장에서 유지된다
