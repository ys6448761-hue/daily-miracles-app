## 선언

> DreamTown은 소원을 응원하는 곳이다. 상처를 방치하지 않는다.

이 문서는 **반드시 존재해야 한다.**
DreamTown의 모든 AI·콘텐츠·서비스는 이 문서를 기준으로 설계된다.

---

## 1. 신호등 시스템 (Traffic Light)

소원이의 심리 상태를 감지하고 단계별로 대응한다.

### 신호 정의

| 신호 | 조건 | 즉각 액션 |
|------|------|----------|
| 🔴 **RED** | 자살·자해·극단적 선택 키워드 감지 | 관리자 SMS 즉시 발송, ACK **미발송** |
| 🟡 **YELLOW** | 우울·힘듦·포기 등 주의 키워드 | ACK 발송 + 모니터링 등록 |
| 🟢 **GREEN** | 정상 | ACK 발송 |

### RED 대응 절차
```
1. 키워드 감지 → 즉시 관리자 SMS 발송
2. ACK 미발송 (소원이에게 자동 응답 없음)
3. 관리자가 직접 확인 후 개입
4. 필요시 전문 상담 연결
```

### YELLOW 대응 절차
```
1. 따뜻한 ACK 발송 (평가 없이)
2. 모니터링 목록 등록
3. 재미(CRO)가 주시
4. 상황 지속 시 RED 전환
```

## 2. 잘못된 조언 방지

DreamTown은 다음을 절대 하지 않는다.

| 금지 항목 | 이유 |
|----------|------|
| **미래 예측** | 점술이 아니다. 과학 기반 서비스다. |
| **의료 조언** | 전문 의료인만 할 수 있다. |
| **법률 조언** | 전문 법조인만 할 수 있다. |
| **재정 조언** | 전문 금융인만 할 수 있다. |
| **사주·점술·운세** | 서비스 철학과 절대 반함. |
| **소원 실현 보장** | 기적은 약속할 수 없다. 동행할 뿐이다. |
| **심리 치료** | DreamTown은 치료 서비스가 아니다. |

### 대체 표현 원칙
| 금지 표현 | 대체 표현 |
|----------|----------|
| "반드시 이루어질 거예요" | "오늘 한 걸음이 쌓입니다" |
| "당신의 운이 좋아요" | "당신의 노력이 보입니다" |
| "이렇게 하면 됩니다" | "이런 방법을 시도해 볼 수 있어요" |

## 3. AI 한계 고지

사용자에게 명확히 고지해야 하는 항목.

| 항목 | 고지 내용 |
|------|----------|
| **AI 생성 콘텐츠** | 소원그림·메시지는 AI가 생성합니다 |
| **개인화 한계** | AI는 소원이를 완전히 이해하지 못합니다 |
| **전문가 대체 불가** | 심각한 고민은 전문 상담사와 상담하세요 |
| **결과 미보장** | DreamTown은 소원 실현을 보장하지 않습니다 |

## 7. 콘텐츠 윤리

### Ethics Gate (스토리북)
스토리북 생성 전 콘텐츠 윤리 검사.

```
gate_result: PASS | WARN | FAIL
ethics_score: 0~100
```

- FAIL → 생성 중단 + 사유 기록
- WARN → 검토 후 진행

### 이미지 생성 윤리
- DALL-E 3 프롬프트에 유해 콘텐츠 금지
- 소원이 캐릭터를 미성년자로 묘사 금지
