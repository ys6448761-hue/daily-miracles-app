# SSOT-PRICE-001 — DreamTown 별빛항로 가격 체계 v1.0

> 이 문서는 별빛항로 상품의 **모든 가격·원가·마진·배분**의 단일 진실 소스(SSOT)다.
> 단가가 바뀌면 **이 문서를 먼저 고치고**, 상품 PDF·견적은 여기서 파생한다.

---

## 0. 가격 3층 구조 (용어 정의)

| 층 | 이름 | 의미 | 노출 대상 |
|---|---|---|---|
| 1 | **정상가** | 할인 전 정가 (표시·취소선용) | 손님 |
| 2 | **판매가** | 실제 손님 통일가 (전 채널 동일) | 손님 |
| 3 | **입금가** | 내부 매입원가 (마진 계산 전용) | 내부 대외비 |

**핵심 원칙**
- 판매가는 **전 채널 통일** (직접·여행사·앤썸 어디서 사든 동일가). 가격 혼선 금지.
- **별빛항로 IP 마진은 (주)여수여행센터 고정 몫.** 모객·진행 주체와 무관하게 양도하지 않는다.
- **여행사 모객 마진은 판매가 인상으로 충당.** 기존 마진 풀(여수·앤썸)을 침해하지 않는다.

## 7. 상품 판매가 (확정)

### 7-1. FIT — 별빛항로 반나절 (Basic)
- 구성: 별빛항로 체험(별씨앗·소원그림·기적영상) + 동행 안내. 식사 자유식. 숙박·교통 없음.
- **판매가 41,000** (정상가 59,800, 약 31%↓)
- 코스: 엑스포역 → 해안산책로 → 이순신광장 → 종포해양공원 → 하멜등대 (약 4시간)

### 7-2. FIT — 1박 2일 · 유탑마리나
- 구성: 유탑 1박 + 케이블카 왕복 + 미남크루즈 + 별빛항로 체험. **식사 자유식.**

| 출발 | 1인 판매가 |
|---|---:|
| 주중(일~목) | **129,000** |
| 금요일 | **169,000** |
| 토요일 | **199,000** |

- 객실 추가: 3인 1실 +10,000 / 4인 1실 +20,000 (1인)
- 일요일은 주중가에 포함(유람선 불꽃 차액 흡수)

### 7-3. FIT — 1박 2일 · 케니 (가성비)
- 구성: 케니 1박 + 케이블카 왕복 + 미남크루즈 + 별빛항로 체험. **식사 자유식.**

| 출발 | 1인 판매가 |
|---|---:|
| 주중(일~목) | **105,000** |
| 금 · 토 | **149,000** |

- 2인실 전용. 일요일은 주중가에 포함.

### 7-4. 단체(5인+) — 1박 2일 · 케니 (확정 기준)
- 기준: **10인 · 스타렉스 · 주중 · 가이드 미적용**
- 구성: 위 FIT + **식사 4끼(91,000) + 차량 + 진행비**

| 구성 | 1인 |
|---|---:|
| 숙박(60,000÷2) | 30,000 |
| 케이블카 왕복 | 15,000 |
| 미남크루즈(주중) | 17,000 |
| 식사 4끼 | 91,000 |
| 차량(스타렉스 ÷10) | 40,000 |
| 별빛항로 (원가) | 25,000 |
| 별빛항로 마진 | 10,000 |
| 진행비 | 15,000 |
| **판매가 (1인)** | **248,000** |

- 별빛항로 B2B가 **35,000** = 원가 25,000 + 마진 10,000
- 가이드 옵션: 400,000/2일 ÷ 인원
- **주말(금·토): 숙박 +25,000(110,000÷2) + 유람선 불꽃 +20,000 = 1인 293,000** (마진 배분은 주중과 동일)

### 7-5. 단체(5인+) — 1박 2일 · 유탑 (확정)
- 케니와 동일 최종 구조(별빛 35,000 / 진행비 5:10 / 차량 앤썸). 케니 대비 **숙박 +17,500 · 식사 +17,000**.

| 출발 | 1인 판매가 | 비고 |
|---|---:|---|
| 주중(일~목) | **282,500** | 248,000 + 34,500 |
| 금요일 | 307,500 | 숙박 금 140,000 반영 |
| 토요일 | 337,500 | 숙박 토 200,000 반영 |

- 10인·스타렉스·주중·가이드 미적용 기준. 유탑 주말은 숙박이 금/토로 갈려 별도 산출.

## 8. 마진 · 수익 배분 구조

### 8-1. 별빛항로 마진 (재료 10,000 + 인건비 15,000 = 원가 25,000 공통)
- **개인·FIT: 판매 41,000 = 원가 25,000 + 마진 16,000.**
- **B2B 단체: 판매 35,000 = 원가 25,000 + 마진 10,000.**
- 인건비 15,000은 푸르미르 노동 대가(향후 채용 시 해당 인력 지급). 마진은 **여수여행센터 고정 IP 마진** — 양도하지 않는다.
- 여행사 모객 시 일부를 모객 수수료로 분배 가능 (배분율 §10 검토).

### 8-2. 단체 수익 배분 (케니 · 10인 · 주중 기준)

| 주체 | 1인 수익 | 구성 |
|---|---:|---|
| **(주)여수여행센터** | **15,000** | 별빛항로 IP 10,000 + 진행비 분할 5,000 (홍보·모객) |
| **여수앤썸** | **29,000** | 호텔차액 10,000 + 레저 6,000 + 식당 빽마진 3,000 + 진행비 10,000 (+차량 운영익) |

### 8-3. 모객 구조별 비교 (케니 · 주중)

| 구조 | 여수여행센터 | 여수앤썸 | 여행사 | 손님가 |
|---|---:|---:|---:|---:|
| 2자 (여수+앤썸, 직접 모객) | 15,000 | 29,000 | — | 248,000 |
| 3자 (+여행사 모객) | 15,000 | 29,000 | 25,000 | **273,000** |

- 여행사 마진 25,000은 **판매가 인상**으로 충당 → 여수·앤썸 몫 불변, 여수여행센터 **최소 15,000 보장.**

## 9. 채널 · 사업 정책

### 9-1. 채널
| 채널 | 여행사 마진 | 비고 |
|---|---:|---|
| 직접 판매 | 0 | 여수여행센터 직접 |
| 여행사 제휴 | **25,000 정액** | 요일·시즌 무관, 판매가 인상으로 충당 |

### 9-2. 사업 분리
| 사업 | 모객 | 진행 | 비고 |
|---|---|---|---|
| 관광 단체 (5~10인 주력) | 여수여행센터 | **여수앤썸**(랜드사) | 관광 포함 시 앤썸 이관 |
| 세미나·학회 (100~1,000인) | 여수여행센터 | **여수여행센터 직접** | 관광 없음 / 별도 모델 (미설계) |

- 관광 단체에 외부 여행사를 추가로 끼우면 마진이 얇아 비권장. 여행사 유통은 **FIT 상품 중심.**
