## 소원이 (Sowoni)

### 정체성
- **역할**: 소원을 품은 모든 사람을 대표하는 메인 내러티브 캐릭터
- **의미**: "소원이" = 소원을 가진 사람 그 자체

### 외형 스펙 (CHARACTER LOCK v2)
| 속성 | 설정 |
|------|------|
| 연령 | 20~22세 한국 대학생 (성인 명시, NOT a minor) |
| 스타일 | 웹툰 큰 눈, 단순한 코(점 또는 선), 부드럽고 둥근 얼굴 |
| 라인 | 단순 애니 선화, 셀 쉐이딩 (3D 옷감 시뮬레이션 금지) |
| 금지 | 교복, 아동 느낌 연출, 옷/소지품에 텍스트 |

### 의상 프리셋
| 코드 | 설명 |
|------|------|
| `SPRING_CASUAL` | 봄 캐주얼 |
| `SUMMER_SEASIDE` | 여름 바닷가 |
| `AUTUMN_COZY` | 가을 아늑함 (니트/편안함) |
| `WINTER_COAT` | 겨울 코트 |
| `NIGHT_WALK` | 밤 산책 |

### 서사 역할
- 소원을 품고 여정을 시작하는 주인공
- 관객(사용자)이 자신을 투영하는 캐릭터
- 아우룸과 함께 Dragon Palace → DreamTown Star 여정을 걷는다

## 아우룸 (Aurum)

### 정체성
- **역할**: 디지털 용궁의 수호자 / 조용한 안내자
- **의미**: Aurum(라틴어 황금) = 소원의 빛, 용궁의 지혜
- **등장 조건**: 전환의 순간에만 나타남 (바다와 별 사이, 소원이 변화하는 순간)

### 외형 스펙 (AURUM TURTLE LOCK v2)
| 속성 | 설정 |
|------|------|
| 형태 | 작고 따뜻한 황금빛 거북 정령 |
| 등껍질 | 둥근 구슬형 몸체, 단순한 scute 판 (6~10개) |
| 얼굴 | 두 개의 작은 눈, 작은 부리형 입, 두 개의 콧구멍 점 |
| 사지 | 짧은 다리, 작은 발톱/지느러미 |
| 특징 | 등껍질에 초승달 문양 1개 (고정 위치) |
| 빛 | 미세한 수채화 헤일로 링 (부피감 있는 빛줄기 금지) |

### 금지 사항
- 액세서리 / 옷처럼 보이는 패턴 금지
- 사람 아기 얼굴 / 속눈썹 / 치아 금지
- 과도한 광택 / 금속성 반사 금지

### 상태 프리셋
| 코드 | 상태 | 사용 씬 |
|------|------|---------|
| `BASE` | 기본 | 일반 등장 |
| `OBSERVE` | 관찰/집중 | 소원을 바라볼 때 |
| `GUIDE` | 미소/인도 | 방향을 알려줄 때 |
| `SPARKLE` | 감동/반짝임 | 소원 성취 순간 |
| `COZY_BREEZE` | 바람 탐 | 전환·휴식 씬 |
| `NIGHT_CALM` | 차분함 | 밤, 성찰 씬 |

### 서사 역할
- 소원이를 직접 이끌지 않고, 곁에서 조용히 빛을 비추는 존재
- 바다(Dragon Palace)와 별(DreamTown Star) 사이의 연결고리
- 말 없이 행동으로 신뢰를 전달한다
