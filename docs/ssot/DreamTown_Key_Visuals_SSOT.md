## 선언

이 문서는 DreamTown의 **반복 사용 핵심 장면**을 정의한다.
아래 장면들은 향후 DreamTown의 모든 브랜딩과 스토리에서 기준 비주얼로 활용된다.

---

## KV-01. 소원이 별이 되는 순간 (The First Wish Becoming a Star)

### 개요

DreamTown의 **첫 기적**을 상징하는 장면.
여수 바다에서 시작된 하늘 속에서 소원은 **별**이 되어 하늘로 떠오른다.
DreamTown의 **창세를 표현하는 핵심 비주얼**로, 소원이·아우룸·거북 별자리가 결합된 마법적 순간.

### 비주얼 요소

| 요소 | 묘사 |
|------|------|
| 소원이 | 바위 위에 앉아 하늘을 응시 |
| 별빛 변환 | 소원이가 별이 되며 하늘로 날아오름 |
| 아우룸 | 거북 별자리와 이어지는 빛의 흐름 생성 |
| 배경 | 여수 바다와 산의 풍경 / 부드러운 수채화 |

### 상징성

| 요소 | 상징 |
|------|------|
| 소원이 → 별 | 꿈과 희망의 실현 |
| 아우룸 | Guide이자 보호자 — 소원의 여정에 동행 |
| 거북 별자리 | DreamTown의 원초적 상징 / 여수 설화 기반 |
| 별빛 | 소원의 가능성과 새로운 시작 |

### 활용

| 채널 | 용도 |
|------|------|
| 영상 | DreamTown 오프닝 영상 |
| 포스터 | DreamTown 테마 포스터 |
| 앱 | DreamTown 소개 화면 |
| 마케팅 | 캠페인 및 프로모션 |

## KV-02. Golden Turtle Constellation (황금거북 별자리)

### 개요

DreamTown의 **상징적 별자리**.
거북이는 여수 설화의 핵심 동물이며, DreamTown의 근본적 가치와 방향성을 나타낸다.

### 비주얼 요소

| 요소 | 묘사 |
|------|------|
| 형태 | 거북이 형상의 별자리가 하늘에 펼쳐짐 |
| 구성 | 별들의 연결선으로 그려짐 |
| 역할 | 빛과 별의 흐름 속에서 소원의 길을 이끄는 중심축 |

### 세계관 연결

```
Golden Nine Constellation = Golden Turtle Constellation의 9개 별
(백도·거문도·사도·금오도·향일암·오동도·제석산·신풍·아우룸)
```

> 상세 구조: `DreamTown_Yeosu_Travel_Routes_SSOT.md`

## KV-03. 아우룸과 북극성 (Aurum and the North Star)

### 개요

아우룸이 북극성을 따라 소원에게 길을 인도하는 장면.
**방향·안내·희망**의 시각적 표현.

### 비주얼 요소

| 요소 | 묘사 |
|------|------|
| 아우룸 | 북극성을 따라 소원에게 길을 인도 / 빛을 나누는 존재 |
| 북극성 (여의보주) | 하늘의 중심에서 소원의 방향을 명확히 비춤 |
| 관계 | 아우룸 → 북극성 → 소원이 (인도의 흐름) |

### 상징성

| 요소 | 상징 |
|------|------|
| 북극성 | 소원이 별이 되는 궁극적 목표 |
| 아우룸 | 길 위의 동행자 |
| 빛의 흐름 | 소원 → 여정 → 실현의 연속 |
