## 향후 적용 원칙

- **기본 문장 1개 고정** — 기본값: `오늘의 마음도 별이 되었어요`
- **감정 상태별 변형** — 3~5개 문장을 감정 분류에 맞게 선택 가능
- **재사용 범위**: 포스트카드 / 카카오 공유 / 별공방 결과 화면
- **신규 문장 추가 시**: 이 SSOT를 먼저 업데이트한 뒤 적용
