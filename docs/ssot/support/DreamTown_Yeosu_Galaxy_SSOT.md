## 선언

> "DreamTown의 하늘은 여수의 바다에서 태어났다."

여수 은하는 DreamTown의 **첫 번째 은하이자 출발점**이다.
모든 소원항해는 여수에서 시작된다.

---

## 1. 여수 은하 개요

| 항목 | 내용 |
|------|------|
| **은하명** | Yeosu Galaxy (여수 은하) |
| **위상** | Origin Galaxy — 모든 DreamTown 은하의 어머니 |
| **현실 위치** | 전라남도 여수시 |
| **상징** | 바다 → 하늘 변환 (Dragon Palace → DreamTown Star) |

---

## 2. 여수가 Origin인 이유

| 상징 요소 | 의미 |
|----------|------|
| 디지털 용궁 (Digital Dragon Palace) | 소원이 빛 구슬로 보관되는 바닷속 공간 |
| 아우룸 (황금 거북 정령) | 바다와 별 사이를 잇는 안내자 |
| 바다 → 별 변환 | 현실 공간에서 소원이 우주로 이어지는 상징 |
| 소원항해단 | 여수에서 출발하는 최초의 드림 보야지 |

---

## 3. 여수 은하 구조

```
Yeosu Galaxy (여수 은하)
      ↓
Yeosu Voyage Route (여수 항해 항로)
      ↓
Courses (탐험 코스)
      ↓
Miracle Points (기적 포인트)
```

---

## 4. 탐험 코스 (4종)

### 코스 구조
```
Location (장소)
      ↓
Emotion (감정)
      ↓
Action (권유 행동)
      ↓
Miracle Moment (기적의 순간)
```
