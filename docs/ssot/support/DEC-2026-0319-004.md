## 결정

별 성장 화면은 통계 대시보드가 아니라
**기록이 별빛으로 축적되는 장면**으로 구현한다.

---

## 고정 사항

### 핵심 철학
> "내가 얼마나 했는지"를 보여주는 곳이 아니라,
> "내 소원이 어떻게 살아남았는지"를 보여주는 곳이어야 한다.
> — 루미

### 시각 표현 원칙
- 성장 = 빛의 층(layer) + 후광(glow) + 잔광(afterglow)의 깊이로 표현
- 기록이 쌓일수록 별빛이 깊어진다 (수치 아님)
- 중앙에 하나의 별, 하단에 최근 기록 3개만 조용히 배치

### 절대 금지
- ❌ 레벨, 점수, 퍼센트 표시
- ❌ streak UI (연속 달성 등)
- ❌ 통계 대시보드 형태
- ❌ 숫자 중심 성장 표현

### 데이터 소스
- `logStorage.getLogs()` — localStorage 기반
- 최근 3개만 렌더링

### 컴포넌트 구조
```
pages/StarGrowth.jsx          ← 화면 진입점
  └─ components/GrowthStar.jsx    ← 중앙 별 (빛 층 표현)
  └─ components/GrowthLog.jsx     ← 최근 기록 3개
```

---

## 이유

- 수치화는 "달성 피로"를 만든다
- 빛의 깊이는 "감각적 축적감"을 만든다
- 3개 제한은 히스토리 부담 없이 현재에 집중하게 한다
