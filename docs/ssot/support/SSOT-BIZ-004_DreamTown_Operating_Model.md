# SSOT-BIZ-004 — DreamTown 운영 모델

## 목적

DreamTown이 고객 경험을 어떻게 운영하는지의 원칙을 고정한다.

---

## 핵심 운영 원칙

DreamTown은 패키지 강제 일정이 아니다.

DreamTown은 **감정 항로를 제공하고, 마지막 선택은 고객에게 맡긴다.**

---

## 하멜등대 이후 자유 선택 원칙

| 항목 | 기준 |
|------|------|
| EP01 공식 프로그램 종료 | **23:30 (하멜등대)** |
| 23:30 이후 | **고객 자율 시간** |
| 강제 일정 | 없음 |
| 24시 이후 공식 일정 | 없음 |
| 추가 소비 유도 | **금지** |

DreamTown은 추가 소비를 유도하지 않는다.
고객은 다음 중 자유 선택한다.

- 호텔 이동
- 낭만포차
- 야간 산책
- 카페
- 자유 휴식

---

## 고객 안내 문구

결제 페이지 / 상품 소개에 반드시 포함한다.

> **별빛 항로는 하멜등대에서 끝나지만,**
> **그 이후의 밤은 당신만의 여수로 이어집니다.**

---

## 감정 항로 설계 원칙

| 원칙 | 내용 |
|------|------|
| 감정 클라이맥스 | 하멜등대 (23:00~23:30) — 변경 불가 |
| 클라이맥스 이후 | 자유 선택 — 강제 일정 없음 |
| 운영 철학 | 회복은 목적지가 아니라 항로다 |

---

## EP01 감정 클라이맥스 고정값

| 항목 | 기준 |
|------|------|
| EP01 감정 클라이맥스 | **하멜등대** (변경 불가) |
| 유람선 역할 | 상승 — 클라이맥스 **진입 전** 감정 준비 구간 |
| 하멜등대 역할 | 소원 / 희망 / 회복 시작 |

> 유람선이 가장 비용이 큰 구간이더라도 감정 클라이맥스는 하멜등대다.
> 운영 편의나 가격 구조로 클라이맥스 위치를 변경하지 않는다.

---

## 운영 우선순위

운영 편의보다 감정 항로를 우선한다.

EP01 기본 순서 (변경 불가):

```
여수엑스포역 → 엑스포 바닷길 → 이순신광장
→ 케이블카 → 유람선 → 종포해양공원 → 하멜등대
```

---

## 자유 선택 연출 금지 사항

다음은 운영 문구·안내에서 사용하지 않는다.

- "~하세요" (지시형)
- "반드시 ~" (강제형)
- 24시 이후 공식 일정 표기
- 호텔 체크인 강제 안내
- 추가 상품·소비 권유 문구
