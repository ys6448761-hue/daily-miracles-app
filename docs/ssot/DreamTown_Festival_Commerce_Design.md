## 핵심 원칙

항해 보급소는 **즉시 소진형이 아니다.**

```
이벤트 = 항해 예약 창을 여는 축제
소원이 = 열린 예약 기간 안에서 자신이 떠날 수 있는 날짜를 선택
```

구매를 강제하는 장치가 아니라, 예약 창을 여는 감정적 트리거다.

---

## 전체 흐름

```
별 탄생
    ↓
광장 축제 (벨 + 빛 + 오로라 + 불꽃)
    ↓
소원이 축하
    ↓
항해 보급소 오픈 (예약 창 열림)
    ↓
소원이가 자신의 날짜 선택
    ↓
여행/체험 예약 완료
```

**감정 → 참여 → 예약** 흐름

## 2. 항해 보급소

### 오픈 타이밍
별 탄생 후 항해 보급소 예약 창 오픈.
소원이는 오픈된 창 안에서 **자신이 떠날 수 있는 날짜를 자유롭게 선택**한다.

> ⚠️ 오픈 시간 미결 — CEO 결정 대기
> 루미 추천: 15분 (긴장감 유지 + 구매 집중 + 이벤트 밀도)
> 대안: 30분

### 세계관 상품명 (일반 쇼핑몰 명칭 금지)

| 상품 유형 | 세계관 명칭 예시 |
|----------|----------------|
| 숙박 특가 | 오늘의 항해 숙소 / 별빛 여행 패스 |
| 체험 연결 | 아우룸 항해권 / DreamTown 투어 패스 |

---

## 3. 상품 유형 3가지

### 1️⃣ 무료 숙박권
```
소량 / 추첨 또는 선착순
예: 제주 별빛 숙박권 / 부산 바다 항해 숙소
```

### 2️⃣ 초특가 숙박
```
여행사 입금가 수준
예: 정가 15만원 → DreamTown 가격 7만원
```

### 3️⃣ 체험 번들
```
숙박 + 소원항해 체험
숙박 + 레저 패스
숙박 + 투어 패스
```

## 4. 마진 구조

```
기존 마진 2.5만원
    ↓ (이벤트 적용)
1.5~1.8만원
    ↓
대량 판매로 보완
```

---

## 5. 환불 정책

```
미사용 시 전액 환불  ← 표시 문구
체험 후 불만족 시 환불
```

제한:
```
사용 후 환불 불가
```

표시 브랜딩:
```
DreamTown 안심 보장
```

---

## 6. "별이 태어난 곳" 연결 구조

```
별 탄생 위치 (예: 여수)
    ↓
해당 지역 항해 상품 자동 연결
    ↓
여수 → 여수 숙박/체험 상품 노출
```

→ DreamTown을 **세계관 여행 플랫폼**으로 만드는 핵심 루프

---

## 7. DreamTown Story Economy 루프

```
소원 → 별 → 축제 → 여행 → 체험 → 새 이야기 → 다시 별
```

이 루프가 완성되면:
- 라이브커머스 + 커뮤니티커머스 + 이벤트커머스 결합
- 광고 없이 이야기 → 감정 → 구매 흐름
