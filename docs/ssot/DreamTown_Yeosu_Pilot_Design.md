## 2. 주간 운영 스케줄

```
수요일 밤 9시   — Super Star Night 축제
        ↓
목요일          — 항해 보급소 예약 창 오픈
        ↓
익일(금요일)~   — 소원이가 자신의 날짜를 선택하는 1개월 예약 창
```

> 즉시 소진형이 아님. 소원이가 예약 창 안에서 자신이 떠날 수 있는 날짜를 자유롭게 선택하는 구조.
> 이벤트는 구매 강제 장치가 아니라 항해 예약 창을 여는 축제 역할.

### 요일별 운영 구분

| 요일 | 운영 방식 |
|------|----------|
| 일~목 | 기본 운영 중심 |
| 금·토 | 한정 노출 또는 프리미엄 상품군 운영 |

## 3. 할인 구조 원칙

```
정면 할인(정가 직접 훼손) 금지
```

대신 아래 4가지 결합:

```
주중 한정 재고
DreamTown 전용 번들
추가 혜택
상한형 지원
```

→ **축제형 혜택 구조**로 운영
