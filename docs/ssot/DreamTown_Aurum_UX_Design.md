## 핵심 원칙

```
아우룸은 항상 "장면의 주인공"이 아니라 "전환의 안내자"로만 등장한다.
```

- 화면 전체를 점유하지 않음
- 필요할 때만 짧게 등장
- 감정이 바뀌는 순간에만 등장

---

## 5단계 등장 동선

### 1단계 — Wish Gate
- **역할**: 입구를 열어줌. 아주 작게 등장.
- **메시지**: "소원을 남겨보세요. 당신의 소원은 혼자가 아닙니다."
- **상태 프리셋**: `GUIDE`
- **주의**: 너무 많이 말하지 않음. 조용히 시작을 열어줌.

### 2단계 — 전달 순간
- **온라인**: 빛구슬을 보낼 때 아우룸이 "받아가는 존재"로 등장
- **오프라인**: 종이비행기를 날릴 때 등장
- **핵심 감정**: "내 소원이 어딘가로 잘 전달되었다"
- **상태 프리셋**: `OBSERVE` → `GUIDE`

### 3단계 — 용궁 보관
- **역할**: 전면 등장 아님. 배경/상징으로 존재.
- **감정**: 안심 / 보관 / 기다림
- **상태 프리셋**: `NIGHT_CALM`

### 4단계 — StarLink 승급
- **역할**: 가장 상징적으로 강한 등장
- **메시지**: "당신의 소원이 별씨앗이 되어 하늘로 오르고 있습니다."
- **상태 프리셋**: `SPARKLE`
- **연출**: 아우룸이 하늘을 바라보는 장면 + Aurora Path 빛줄기

### 5단계 — DreamTown 탐험
- **역할**: 조용한 길잡이
- **메시지 예시**:
  - "오늘의 울림으로 가볼까요"
  - "당신의 별과 닿아 있는 이야기가 있어요"
  - "이 별자리를 따라가 보면 비슷한 항해를 만날 수 있어요"
- **상태 프리셋**: `COZY_BREEZE`

## 4가지 UI 모듈

### 1. 아우룸 안내 버블
짧은 한 줄 메시지

예시:
```
소원을 남겨보세요
용궁에 안전하게 보관되었어요
별이 될 준비를 하고 있어요
```

### 2. 아우룸 전달 애니메이션
빛구슬 / 종이비행기를 받거나 인도하는 장면 (Lottie or CSS)

### 3. 아우룸 용궁 스탬프
오프라인 체험 결과물에 들어가는 상징

```
용궁 보관 완료
아우룸 전달 확인
StarLink 대기 중
```

### 4. 아우룸 탐험 가이드
DreamTown 안에서 추천 길 제시

```
비슷한 별자리 보러 가기
오늘의 지혜 보러 가기
내 별과 연결된 이야기 보기
```

---

## 등장 금지 조건

- 광고/구매 유도 장면
- 에러/오류 화면
- 로딩 중 (세계관 희석)
- 같은 화면에 2회 이상 반복 등장
