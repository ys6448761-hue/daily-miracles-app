## 핵심 선언

```
DreamTown의 하늘은 고정된 하늘이 아닙니다.
새로운 소원이 별이 되고, 비슷한 항해가 별자리가 되며,
많은 별자리가 하나의 은하를 이루고,
은하들이 모여 더 큰 지혜의 우주로 확장됩니다.
DreamTown은 완성된 하늘이 아니라 계속 자라는 우주입니다.
```

DreamTown은 사람들의 소원이 별이 되고,
별들이 별자리가 되며, 별자리가 은하가 되고,
은하들이 다시 더 큰 지혜의 우주로 확장되는
**Human Wisdom Universe**다.

---

## 1. 4단계 확장 계층

```
Star (별)               ← 한 소원이의 이야기
        ↓
Constellation (별자리)  ← 비슷한 항해 패턴 묶음
        ↓
Galaxy (은하)           ← 여러 별자리의 주제 묶음
        ↓
Galaxy Cluster (은하군) ← 은하보다 큰 가치 축
        ↓
Cosmic Expansion        ← 계속 자라는 우주
```

## 4. 동서남북 기본 좌표계

DreamTown 우주의 나침반. 우주가 커질수록 더 중요해짐.

| 방향 | 테마 |
|------|------|
| 북쪽 | 도전 / 창조 / 시작 |
| 서쪽 | 사랑 / 관계 / 연결 |
| 동쪽 | 성장 / 배움 / 발견 |
| 남쪽 | 치유 / 회복 / 균형 |

사용자는 항상 이 질문으로 돌아올 수 있다:
> "나는 지금 어떤 항해를 찾고 있는가?"

## 5. 은하 화면 UX 구조

은하에 들어가면 목록이 아닌 **행성 탐험** 느낌으로.

### 예: 창업 은하

```
은하 소개
→ 새로운 시작과 도전, 실패와 재도전의 항해가 모인 곳

대표 항로
→ 준비 → 첫 시도 → 실패 → 재정비 → 첫 성공

주요 감정
→ 두려움 / 용기 / 고독 / 성취

대표 별자리
→ 첫 창업 항해 / 작은 가게의 빛 / 첫 고객의 순간

탐험 방법
→ 대표 별자리 보기
→ 최근 떠오른 별 보기
→ 가장 울림이 큰 별 보기
→ 나와 비슷한 항해 찾기
```

## 6. 탐색 레이어 (5축)

우주가 커질수록 탐색 레이어 필수.

| 축 | 탐색 방법 |
|----|----------|
| 방향 탐색 | 동 / 서 / 남 / 북 |
| 우주 레벨 탐색 | 별 / 별자리 / 은하 / 은하군 |
| 감정 탐색 | 희망 / 용기 / 위로 / 감사 / 회복 |
| 지역 탐색 | 여수 / 서울 / 제주 / 글로벌 |
| 시간 탐색 | 최근 / 오래된 / 계절별 |

## 7. 글로벌 확장 구조

```
작은 하늘 (여수)
    ↓
지역 하늘 (한국)
    ↓
언어권 하늘 (아시아)
    ↓
글로벌 하늘 (전 세계)
```

```
Korea Sky
Japan Sky
English Sky
Spanish Sky
Global Shared Sky
```

각 하늘은 완전 분리가 아니라 큰 우주 안에 공존.
→ **여러 문화권의 소원이 공존하는 다층 우주**
