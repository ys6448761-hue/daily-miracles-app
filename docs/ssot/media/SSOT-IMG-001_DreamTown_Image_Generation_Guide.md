# SSOT-IMG-001 — DreamTown 이미지 생성 가이드

## 목적

DreamTown의 감정 전달은 영상보다 이미지에서 시작된다.

> EP01 발견: 영상 감정 전달력 = 이미지 70% + 영상 20% + 편집 10%

모든 이미지 생성은 본 문서를 따른다.
도구(ChatGPT / Midjourney / 기타)가 바뀌어도 이 기준은 유지된다.

---

## 1. DreamTown 이미지 원칙

DreamTown은 관광 홍보 이미지가 아니다.

DreamTown은

> **소원 → 감정 → 회복**

을 표현하는 이미지다.

### 이미지가 통과해야 하는 3가지 질문

1. 감정이 먼저 오나?
2. 소원이가 공간 안에 자연스럽게 있나?
3. 여수가 배경이 아닌 감정의 그릇이 되나?

---

## 2. 비주얼 스타일

### 고정

| 항목 | 기준 |
|------|------|
| 표현 | 2D |
| 스타일 | 한국만화 / 지브리 감성 |
| 기법 | 수채화 |

### 금지

- 3D 렌더링
- 실사 사진
- 과도한 디테일
- AI 렌더링 느낌 (plasticity, 과도한 광택)

---

## 3. 소원이 구성

### 고정 요소 (항상 유지)

- 긴 검은 머리
- 아이보리 계열 상의
- 밝은 파란 계열 의상
- 작은 숄더백
- 조용한 분위기

### 금지

- 과한 표정 변화
- 과한 액션 포즈
- 틱톡·SNS 스타일 리액션

---

## 4. 카메라 (시점) 원칙

DreamTown 이미지는 관찰자 시점으로 설계한다.

| 허용 | 제한 |
|------|------|
| 옆모습 | 정면 응시 (최소화) |
| 45도 시선 | 카메라를 직접 보는 구도 |
| 뒷모습 | — |
| 위에서 내려다보기 | — |

> 소원이가 관객을 보는 것이 아니라, 관객이 소원이의 여정을 따라가는 구조다.

---

## 5. 배경 원칙

- 소원이보다 **공간이 더 크다**
- 여수 실존 장소 기반 (임의 장소 생성 금지)
- 여수가 배경이 아니라 **감정을 품는 그릇**이 된다

### 우선 장소

1. 여수엑스포역
2. 엑스포 바닷길
3. 이순신광장
4. 케이블카
5. 종포해양공원
6. 하멜등대
7. 모이핀
8. 오동도
9. 장도

---

## 6. 감정 설계 — 버전별 기준

### Original 버전 (EP01 기준)

| 감정 | 수치 |
|------|------|
| 슬픔 | 60 |
| 희망 | 40 |

**분위기:** 바람에 흔들리는 머리카락, 먼 곳을 바라보는 시선, 어깨가 약간 내려앉음

---

### Positive 버전 규칙

| 감정 | 수치 |
|------|------|
| 슬픔 | 40 |
| 희망 | 60 |

**적용 요소:**
- 아주 작은 미소 (과하지 않게)
- 수평선 바라보기
- 어깨 펴기
- 따뜻한 광원 (역광 또는 옆 빛)

> Positive 버전은 Original의 감정을 **지우지 않는다.**
> 슬픔 위에 희망이 올라오는 구조다.
