# SSOT-LOC-001 — DreamTown 여수 장소 SSOT

## 목적

DreamTown에서 장소는 단순 관광지가 아니다.

각 장소는 소원이의 감정 여정에서 고유한 역할을 수행한다.

> **장소도 캐릭터다.**

장소의 감정 역할은 변경하지 않는다.
새로운 장소가 추가되더라도 기존 장소의 역할을 침범하지 않는다.

---

## 장소 감정 역할 정의

### 1. 여수엑스포역

| 항목 | 내용 |
|------|------|
| 역할 | 시작 |
| 감정 | 도착 / 낯섦 / 기대 |
| 소원이 상태 | 현실 세계에서 여수로 진입. 아직 마음을 열지 못함. |
| 영상 키워드 | 캐리어 / 첫걸음 / 역사 / 햇살 |

---

### 2. 엑스포 바닷길

| 항목 | 내용 |
|------|------|
| 역할 | 전환 게이트 |
| 감정 | 긴장 완화 / 호흡 회복 |
| 의미 | 서울의 속도 → 여수의 속도 변환 지점 |
| 영상 키워드 | 바람 / 야자수 / 바다 / 산책 |

---

### 3. 이순신광장

| 항목 | 내용 |
|------|------|
| 역할 | 연결 |
| 감정 | 도시의 온기 / 사람의 온기 |
| 의미 | 혼자가 아니라는 감각 |
| 영상 키워드 | 노을 / 광장 / 도시 / 걷기 |

---

### 4. 케이블카

| 항목 | 내용 |
|------|------|
| 역할 | 상승 |
| 감정 | 희망 / 확장 |
| 의미 | 시야가 넓어짐 |
| 영상 키워드 | 하늘 / 야경 / 빛 / 상승 |

---

### 5. 종포해양공원

| 항목 | 내용 |
|------|------|
| 역할 | 쉼 |
| 감정 | 정리 / 고요함 |
| 의미 | 생각을 내려놓는 장소 |
| 영상 키워드 | 벤치 / 바다 / 침묵 / 파도 |

---

### 6. 하멜등대

| 항목 | 내용 |
|------|------|

| 감정 | 소원 / 별 |
| 의미 | 소원이 별이 되는 장소. **DreamTown 핵심 장소.** |
| 영상 키워드 | 등대 / 별빛 / 밤바다 / 희망 |

---

### 7. 호텔

|------|------|

| 의미 | 마음을 내려놓는 공간 |
| 영상 키워드 | 창문 / 커튼 / 야경 / 침대 |

## 장소 감정 진행 구조

```
여수엑스포역   → 도착
     ↓
엑스포 바닷길  → 호흡
     ↓
이순신광장     → 연결
     ↓
케이블카       → 상승
     ↓
종포해양공원   → 쉼
     ↓
하멜등대       → 소원
     ↓
호텔           → 안식
```

---

## 변경 금지 원칙

- 각 장소의 감정 역할은 변경하지 않는다
- 장소 순서를 역행하는 구성은 금지한다 (예: 하멜등대 → 엑스포역 불가)
- 새로운 장소 추가 시 기존 7개 장소의 역할을 침범할 수 없다
- 신규 장소 추가는 본 문서를 업데이트해야 한다
