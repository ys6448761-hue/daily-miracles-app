# SSOT-DESIGN-001 — DreamTown 역순 감정 설계

## 목적

DreamTown은 역순 감정 설계(Reverse Emotional Design)를 따른다.

항로의 시작이 아닌 **마지막 감정**에서 설계를 시작한다.

> 목적지를 먼저 정한 뒤, 그곳에 이르는 길을 역산한다.

---

## 일반 제작 순서 vs DreamTown 제작 순서

### 일반 제작 순서

```
장소 선정
  ↓
이미지 제작
  ↓
영상 제작
  ↓
감정 확인
```

### DreamTown 제작 순서

```
마지막 감정 정의
  ↓
엔딩 문장 정의
  ↓
감정 항로 역산
  ↓
장소 배치
  ↓
이미지 제작
  ↓
영상 제작
```

---

## EP01 적용 사례

### EP01 최종 감정 (엔딩)

> "여수에서, 내 소원이 다시 빛나기 시작했다."

### EP01 역순 구조 (설계 방향)

```
안식 (이 감정에서 시작해서 역산)
  ←
소원
  ←
쉼
  ←
상승
  ←
연결
  ←
호흡
  ←
도착
```

### EP01 정방향 구조 (실제 항로)

```
도착
  ↓
호흡
  ↓
연결
  ↓
상승
  ↓
쉼
  ↓
소원
  ↓
안식  ← 여기서 "다시 빛나기 시작"
```

## 에피소드별 엔딩 감정 기준

| 에피소드 | 엔딩 문장 | 엔딩 감정 수치 |
|---------|----------|--------------|
| EP01 | 여수에서, 내 소원이 다시 빛나기 시작했다. | 희망 65 |
| EP02 | 마음이 열리기 시작했다. 아직 작은 문이지만. | 희망 60 (목표) |
| EP03 | 처음으로, 소원을 소리 내어 말할 수 있었다. | 희망 70 (예정) |

---

## 역순 설계 적용 규칙

1. 모든 새 에피소드는 **엔딩 문장을 먼저 작성**한다
2. 엔딩 문장에서 역산하여 항로를 배치한다
3. 항로가 확정된 뒤 이미지·영상 제작을 시작한다
4. 이미지 생성 프롬프트에는 해당 장면의 감정 수치를 명시한다
