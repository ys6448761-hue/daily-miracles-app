Version: v1.0
Owner: Aurora5 / 루미
Status: Confirmed
Purpose: 별 탄생 수 제한, 선정 기준, 이벤트 연출, 성장 로드맵

Last Updated: 2026-03-09
Updated By: Code (Claude Code)

---

## 핵심 원칙

```
DreamTown 하늘에는 하루에 태어날 수 있는 별의 수가 제한됩니다.
```

목적: 별의 가치 유지 + 광장 이벤트 집중도 + 스토리 품질 보호
→ DreamTown이 SNS처럼 변하지 않도록 하는 운영 장치

---

## 1. 별 탄생 수 기준

| Phase | 하루 별 탄생 수 | 상징 |
|-------|--------------|------|
| Phase 1 (초기) | **12개/일** | 별자리 12 / 우주 상징 |
| Phase 2 | 24개/일 | 하루 24시간 |
| Phase 3 | 48개/일 | 성장 단계 |

**초기값: 12 Stars / Day** (희소성이 핵심)

---

## 2. 별 탄생 흐름

```
용궁 저장
    ↓
별씨앗
    ↓
StarLink 후보
    ↓
선정 (아래 기준 적용)
    ↓
DreamTown 별 탄생
    ↓
광장 이벤트 (21:00)
```

모든 소원이 별이 되는 것이 아니라 **선별된 소원**만 별이 된다.

---

## 3. 별 선정 기준 (StarLink 승급 조건)

| 기준 | 내용 |
|------|------|
| 스토리 구조 | 소원 → 항해 → 깨달음 → 나눔 구조가 완성된 이야기 |
| 공감 반응 | 기적나눔✨ / 지혜나눔🧠 반응을 받은 이야기 |
| 큐레이션 | DreamTown Guide(Aurora5)가 선정 |

---

## 4. 별 탄생 시간

```
매일 21:00 — DreamTown Night Event
```

이 시간에만 별이 탄생한다. 매일 밤 의식(Ritual)처럼 운영.

### 이벤트 연출 순서
```
DreamTown 하늘 어두워짐
        ↓
Aurora 빛 등장
        ↓
빛구슬 상승
        ↓
별 생성
```

메시지:
```
✨ 오늘의 별이 떠올랐습니다
```

### 아우룸 등장
별 탄생의 증인으로 등장:
```
바다 위에서 하늘을 바라보는 장면
```

---

## 5. First Star Ceremony

사용자의 첫 별 탄생 시 특별 연출:

```
✨ 당신의 첫 별이 DreamTown에 떠올랐습니다
```

→ 사용자에게 강한 감정 경험 제공
→ 바이럴 공유 포인트

## 7. 전략 효과

| 효과 | 내용 |
|------|------|
| 별 가치 상승 | 별이 희소해 의미 있음 |
