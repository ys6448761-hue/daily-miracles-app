## 선언

DreamTown IP 보호는 **4단계 레이어 전략**으로 접근한다.
세계관·캐릭터·시스템·브랜드를 각각 다른 IP 수단으로 중첩 보호한다.

> 법률 전문가 검토 전 참고용 전략 문서.
> 실제 출원/등록 전 IP 전문 변호사 상담 필수.

## 2. 보호 수단별 설명

### 상표권 (Trademark)
- **출원처**: 특허청 (KIPRIS)
- **분류**: 제9류(소프트웨어), 제41류(교육/엔터), 제42류(IT서비스)
- **우선 대상**: "DreamTown", "하루하루의 기적", "아우룸", "소원이"
- **비용 (추정)**: 1건 60~100만 원 / 분류당
- **보호 기간**: 10년 (갱신 가능)

### 저작권 (Copyright)
- **자동 발생**: 창작과 동시에 권리 발생 (등록 불필요)
- **등록 권장**: 한국저작권위원회 등록 → 분쟁 시 증거력 강화
- **주요 대상**: 캐릭터 일러스트, 세계관 텍스트, 스토리보드
- **보호 대상 아닌 것**: 아이디어 자체, 설화 원본 (공공 영역)

### BM특허 (Business Method Patent)
- **출원처**: 특허청
- **적용 가능 시스템**:
  - 소원 입력 → 기적지수 산출 → 별 성장 알고리즘
  - 신호등 기반 위기 감지 + 자동 대응 시스템
  - 소원 추적 Day7/30/90 자동화 시스템
- **비용 (추정)**: 출원 200~400만 원 (변리사 수임료 포함)
- **심사 기간**: 약 1.5~2년

### 디자인권 (Design Patent)
- **출원처**: 특허청
- **대상**: 아우룸·소원이 캐릭터 외형 (확정 버전 기준)
- **조건**: 최종 확정된 디자인이 있어야 출원 가능
- **보호 기간**: 20년

### 영업비밀 (Trade Secret)
- **보호 방법**: 비공개 유지 + 내부 NDA
- **대상**: Aurora5 내부 알고리즘, 메시지 로직, 프롬프트 상세

## 3. 우선순위 로드맵

### Phase 1 — 즉시 (현재)
```
✅ 저작권 자동 보호 활용
   → 모든 창작물에 ⓒ 2026 DreamTown. All rights reserved. 표기
   → 코드/문서 헤더에 저작권 표시 추가

✅ 영업비밀 관리
   → 내부 알고리즘 비공개 유지
   → Aurora5 팀원 NDA 검토
```

### Phase 2 — 3개월 내
```
⬜ 상표권 출원 (최우선)
   → "DreamTown" 제9류·제41류·제42류
   → "하루하루의 기적" 동일 분류
   → 예산: 약 300~600만 원

⬜ 캐릭터 저작권 등록
   → 아우룸 최종 버전 확정 후 즉시
   → 소원이 최종 버전 확정 후 즉시
```

### Phase 3 — 6개월 내
```
⬜ BM특허 출원
   → 기적지수 산출 시스템
   → 소원 추적 자동화 시스템
   → 변리사 상담 필수

⬜ 디자인권 출원
   → 캐릭터 디자인 확정 후
```

### Phase 4 — 1년 내
```
⬜ 해외 상표 출원
   → PCT(국제) 또는 주요 국가별
   → 일본·미국·동남아 순
```

## 4. 여수 금오설화 관련 법적 고려

### 원본 설화
- 여수 금오설화는 **공공 영역(Public Domain)** — 보호 불필요, 누구나 사용 가능
- DreamTown의 창작 재해석 부분만 저작권 보호 대상

### 핵심 원칙
```
원본 설화 = 공공 영역 (자유 사용)
DreamTown 재해석 = 저작권 보호 대상
```

### 필수 고지 문구 (모든 콘텐츠에 포함)
```
DreamTown's origin myth is inspired by the local legends of Yeosu, South Korea.
This is a creative reinterpretation and not an official representation of Yeosu City.
```
