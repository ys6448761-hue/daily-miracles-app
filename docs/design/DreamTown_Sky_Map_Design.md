## 스타일 정의

### 천문 레이어 (기반)
- 실제 밤하늘처럼 별의 밝기·거리감·깊이감 표현
- 별자리 연결선 존재
- 은하수 질감 배경

### 신화 레이어 (세계관)
- 수채화 질감 오버레이
- 별자리 영역마다 신화적 색상 톤 차별화
- 아우룸이 지도 위를 천천히 움직임
- Golden Nine Constellation은 특별한 황금빛 강조

### 합성 결과
```
실제 밤하늘의 깊이감
+
지브리 수채화의 따뜻함
+
한국 신화의 황금빛
```
