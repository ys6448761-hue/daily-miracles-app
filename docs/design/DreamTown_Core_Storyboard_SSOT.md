## 선언

이 문서는 DreamTown 세계관의 **핵심 장면 5개**를 정의한다.
브랜드·앱·영상·마케팅이 모두 이 5장면을 공유한다.

## 세계관 흐름

```
Scene 1: Wish      — 소원을 남기는 순간
    ↓
Scene 2: Star      — 첫 별 탄생
    ↓
Scene 3: Aurum     — 안내자 등장
    ↓
Scene 4: Constellation — 거북 별자리
    ↓
Scene 5: DreamTown — 세계의 탄생
```

## Scene 1. 첫 소원 — Wish Gate

**소원을 남기는 순간**

### 장면

```
여수 바다 위.

한 사람이 처음으로 소원을 남긴다.

작은 빛이 손 위에서 조용히 태어난다.
```

### 핵심 상징

```
사람 → 소원 → 빛
```

### 앱 연결

| 장면 | 앱 화면 |
|------|---------|
| 소원을 남기는 순간 | 소원 입력 화면 (Wish Gate) |

### 비주얼 키워드
```
여수 바다
손 위의 작은 빛
수채화 밤하늘
조용한 시작
```

---

## Scene 2. 첫 별 탄생 — The First Star

**소원이 별이 되는 순간**

### 장면

```
소원이 하늘로 떠오른다.

빛은 점점 커지고

별이 된다.
```

### 핵심 메시지

> **소원은 언젠가 별이 된다.**

### 비주얼 키워드
```
빛이 하늘로 상승
별로의 변환
soft glow / watercolor
희망의 순간
```

---

## Scene 3. 아우룸 등장 — Aurum the Guide

**안내자의 등장**

### 장면

```
별빛 속에서 작은 거북이 나타난다.

아우룸.

그는 말하지 않는다.

그저 길을 비춘다.
```

### 역할 구조

```
주인공 = 소원 (소원이)
아우룸 = 안내자 (Guide)
```

### 아우룸 철학

- 경험을 지배하지 않는다
- 짧게 나타났다가 사라진다
- 길을 비출 뿐, 방향을 강요하지 않는다

### 비주얼 키워드
```
작은 황금 거북
별빛 속 등장
soft warm glow (#F4C542)
조용한 존재감
```

---

## Scene 4. 거북 별자리 — Golden Turtle Constellation

**별들이 연결되는 순간**

### 장면

```
별들이 연결된다.

그리고

거북 별자리가 나타난다.
```

### 상징 구조

```
별  = 소원들
별자리 = 소원들이 만드는 길
```

### 세계관 연결

```
Golden Nine Constellation
= 9개의 별 (백도/거문도/사도/금오도/향일암/오동도/제석산/신풍/아우룸)
```

> 상세: `DreamTown_Yeosu_Travel_Routes_SSOT.md`

### 비주얼 키워드
```
별들의 연결선
거북 실루엣 형성
night sky / watercolor
DreamTown의 지도
```

---

## Scene 5. DreamTown 탄생 — The Galaxy

**세계의 탄생**

### 장면

```
별들이 모인다.

그리고

하늘에 하나의 세계가 만들어진다.

DreamTown.
```

### 핵심 정의

```
DreamTown
= 사람들의 소원이 모여 만들어진 세계
```

### 비주얼 키워드
```
은하 전체 뷰
작은 빛들의 집합
마을과 하늘의 경계
따뜻한 보라색 팔레트 (#9B87F5)
```
