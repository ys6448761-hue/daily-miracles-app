## Scene 1 — 여수 밤바다 (0~5초)

```
화면: 완전한 어둠
        ↓
여수 밤바다가 천천히 드러남
잔잔한 파도, 수채화 질감
하늘에 별 몇 개
```

**카메라**: 수면 위 낮은 앵글 → 천천히 위로 틸트
**색상**: 딥 네이비 (#1a1f3c) + 수면 반사 빛
**사운드**: 파도 소리 (아주 작게) + 피아노 첫 음

**텍스트**: 없음

---

## Scene 2 — 빛구슬 탄생 (5~10초)

```
수면 위에서
작은 빛 하나가 조용히 떠오름

빛구슬
(누군가의 소원)
```

**카메라**: 빛구슬을 따라 천천히 줌인
**색상**: 소프트 골드 (#F4C542) 빛구슬, 주변 수채화 번짐
**사운드**: 피아노 멜로디 전개 + 작은 차임 소리

**텍스트** (빛구슬 아래, 작은 폰트, 페이드인):
```
누군가의 소원이 빛이 되었습니다
```

---

## Scene 3 — 아우룸 등장 (10~16초)

```
빛구슬 옆으로
작은 황금 거북이 조용히 나타남

아우룸.

그는 말하지 않는다.
그저 빛구슬을 바라본다.
```

**카메라**: 아우룸과 빛구슬 함께 프레임 → 아우룸 눈 클로즈업
**색상**: 아우룸 황금 (#F4C542), 부드러운 글로우
**사운드**: 멜로디 따뜻해짐, 첼로 한 줄 추가

**텍스트** (아우룸 등장 직후, 페이드인):
```
아우룸이 소원을 받았습니다
```

---

## Scene 4 — 디지털 용궁 (16~22초)

```
아우룸이 빛구슬과 함께
수면 아래로 내려간다

깊은 바닷속
황금빛 용궁이 드러남

수백 개의 빛구슬이
조용히 빛나고 있다
```

**카메라**: 수면 아래로 하강 → 용궁 전체 풀샷
**색상**: 딥 틸 (#0d3b4f) 바다 + 골드 용궁 + 수백 개 소프트 빛구슬
**사운드**: 음악 깊어짐, 오케스트라 스웰

**텍스트** (용궁 전체가 보이는 순간):
```
모든 소원이 여기 보관됩니다
```

---

## Scene 5 — Aurora Path + 별 탄생 (22~27초)

```
용궁에서
빛줄기 하나가 하늘로 솟구침

Aurora Path.

수면을 뚫고
하늘로 이어지는 빛의 길

그리고

별 하나가 탄생한다
```

**카메라**: 빛줄기를 따라 고속 상승 → 하늘에서 별 탄생 광각
**색상**: 오로라 그린-퍼플 (#9B87F5, #6fcf97) 빛줄기 → 별 골드
**사운드**: 음악 클라이맥스, 차임 + 오케스트라 풀

**텍스트** (별 탄생 순간):
```
소원이 별이 되었습니다
```

---

## Scene 6 — DreamTown 타이틀 (27~30초)

```
카메라가 하늘로 올라감

수많은 별들이 보임

그 별들 사이에서

DreamTown
```

**카메라**: 하늘 광각 → 별들 가득한 하늘 + 타이틀 페이드인
**색상**: 딥 퍼플 밤하늘 (#1a1040) + 별들 소프트 골드
**사운드**: 음악 잔잔하게 마무리, 마지막 피아노 음

**텍스트**:
```
DreamTown

당신의 소원이 별이 되는 곳
```

**CTA** (타이틀 직후):
```
[ 소원 남기기 ]
```

## 사운드 디자인

| 구간 | 사운드 |
|------|--------|
| 0~5초 | 파도 소리 + 피아노 첫 음 (pp) |
| 5~10초 | 피아노 멜로디 + 차임 |
| 10~16초 | 멜로디 + 첼로 (따뜻함) |
| 16~22초 | 오케스트라 스웰 (깊음) |
| 22~27초 | 클라이맥스 (차임 + 풀 오케스트라) |
| 27~30초 | 잔잔한 마무리 + 피아노 |

**금지**: 드롭 비트, 강한 퍼커션, 전자음
**지향**: 지브리 OST + 한국 전통 악기 1가지 (가야금 or 해금) 혼합
