## 핵심 원칙

```
정률 할인 지원이 아니라 상품군별 지원 상한 + 노출 가치 모델
```

```
최종 할인 = 업체 할인 + DreamTown 지원 (공동)
```

---

## 1. 상품군별 지원 상한

### 숙박 상품
원가율이 높아 지원 상한을 낮게 설정.

| 항목 | 내용 |
|------|------|
| DreamTown 지원 상한 | 1객실당 최대 3~5만원 |
| 구조 | 호텔 자체 할인 + DreamTown 지원 |

예시:
```
정가 20만원
호텔 할인 6만원
DreamTown 지원 3만원
→ 판매가 11만원
```

### 체험 상품
마진이 숙박보다 높음.

| 항목 | 내용 |
|------|------|
| DreamTown 지원 상한 | 1인당 최대 1~2만원 |

예시:
```
요트 체험 8만원
업체 할인 2만원
DreamTown 지원 1만원
→ 판매가 5만원
```

### 특산품
마진 구조가 다양 → 지원보다 노출 + 번들 우선.

| 항목 | 내용 |
|------|------|
| DreamTown 지원 상한 | 상품가의 10~15% 한도 |

예시:
```
여수 특산품 세트 4만원
업체 할인 8천원
DreamTown 지원 5천원
→ 판매가 약 2.7만원
```

## 2. DreamTown 지원 방식 3가지

현금 지원 단독 금지. 아래 3가지를 조합.

### 1️⃣ 가격 지원
DreamTown 할인 직접 지원

### 2️⃣ 노출 지원
```
광장 메인 노출
Super Star Night 대표 상품
오늘의 별빛 특가
```

### 3️⃣ 번들 지원
상품을 묶어 가치 상승:
```
숙박 + 체험
숙박 + 특산품
체험 + 특산품
```

---

## 3. 파트너 등급 시스템

### A 파트너 (핵심)
대표 호텔 / 대표 체험 / 대표 특산품

혜택:
```
Super Star Night 대표 노출
DreamTown 지원 우선
```

### B 파트너
일반 숙박 / 체험 상품 / 특산품

혜택:
```
일반 노출
번들 참여
```

### C 파트너
소량 상품 / 시험 상품

혜택:
```
노출 중심
```

