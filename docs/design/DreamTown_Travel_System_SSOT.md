## 핵심 개념

> **여행 → 인생항로 (Travel → Life Voyage)**

DreamTown에서 여행은 단순한 관광이 아니다.
소원이가 소원을 품고 걷는 **인생의 항로**다.

---

## 1. 항로 체계

```
은하 항로 (Galaxy Route)
      ↓
여행 일정 (Itinerary)
      ↓
감정 기록 (Emotion Log)
      ↓
성장 기록 (Growth Record)
```

---

## 2. 항로 4종 (여수 기준)

### 주중 항로 (Weekday Route)
| 항목 | 내용 |
|------|------|
| **대상 요일** | 월~목, 일 (`mon-thu`, `sun`) |
| **포지션** | 가성비, 한적함, 커플·소그룹 |
| **호텔 추천** | 케니(2~3인), 라마다(2~4인) |
| **레저** | 케이블카 ₩16,000 / 유람선 ₩13,000 / 요트 ₩35,000 |
| **코스 연결** | 오동도 숨결, 케이블카 시야 |

### 별빛 항로 (Starlight Route)
| 항목 | 내용 |
|------|------|
| **대상 요일** | 금·토·공휴일 (`fri`, `sat`, `holiday`) |
| **포지션** | 프리미엄, 불꽃 감성, 기념일 |
| **호텔 추천** | 유탑마리나(마리나뷰), 오동재(공휴일 특화) |
| **레저** | 불꽃유람선 ₩35,000 / 불꽃요트 ₩55,000 |
| **코스 연결** | 돌산 별빛 |

### 소망 항로 (Wish Route)
| 항목 | 내용 |
|------|------|
| **대상** | 소원항해단 체험 상품 |
| **포지션** | Daily Miracles 고유, 소원 실현 체험 |
| **상품** | Basic ₩34,900 / Online ₩24,900 / Experience ₩35,000 |
| **코스 연결** | 향일암 일출 |
| **장소 정책** | 주말·공휴일 → 갤러리/몽돌 / 평일 → 배/갤러리 |

### 패밀리 항로 (Family Route)
| 항목 | 내용 |
|------|------|
| **대상** | 3~4인 가족 |
| **호텔 추천** | 오동재 4인 |
| **레저** | 아쿠아플라넷 + 유람선 |
| **예상 합계** | ₩320,000 / 4인 |

## 3. 호텔 포지셔닝

```
가성비 ──── 케니 ─── 라마다 ─── 유탑마리나 ─── 오동재 ──── 프리미엄
           2~3인     2~4인     2~4인 (마리나)   2~4인 (공휴일)
```

| 호텔 | 평일 2인 | 금·토 2인 | 공휴일 2인 |
|------|----------|----------|-----------|
| 케니 | ₩50,000 | ₩70,000 | ₩100,000 |
| 라마다 | ₩70,000 | ₩90,000 | ₩200,000 |
| 유탑마리나 | ₩75,000 | ₩135,000 | ₩175,000 |
| 오동재 | ₩80,000 | ₩110,000 | ₩210,000 |

## 4. 여행 일정 자동 생성

> 출처: `services/itineraryService.js`

| 조건 | 처리 방식 |
|------|----------|

### 이동수단

## 5. 감정 기록 (Emotion Log)

여행 중 느낀 감정을 기록한다.

```
장소 + 감정 + 행동 = 기적 카드
```

여행에서 발생한 기적은 `source: galaxy_exploration`으로 저장된다.

> 기적 카드 구조: `DreamTown_Miracle_System_SSOT.md`

---

## 6. 성장 기록 (Growth Record)

여행이 소원이의 별 성장에 기여하는 방식.

| 여행 행동 | 별 성장 기여 |
|----------|------------|
| 코스 탐험 완료 | 기적 포인트 획득 |
| 기적 카드 기록 | 별 에너지 +1 |
| 소망 항로 완주 | 별 단계 전환 |
| 여수3합 패스 사용 | 커뮤니티 연결 |
