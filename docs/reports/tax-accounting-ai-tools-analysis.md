## 3. 도구별 상세 분석

### 3.1 캐시노트 (한국신용데이터)

**개요**: 전국 160만 사장님이 사용하는 경영관리 앱

**장점**:
- 기본형 **완전 무료**
- 카드매출/입금 자동 수집
- 현금영수증/세금계산서 조회
- AI 비서 '캐시니' 제공 (2025년)
- K-비대면바우처 90% 지원 가능

**단점**:
- 외부 API 미공개
- 세금 신고 기능 없음 (조회만)
- 세무사 연동 별도

**적합도**: ⭐⭐⭐⭐ (매출/입금 관리용)

> 출처: [캐시노트 공식](http://info.cashnote.kr/), [캐시노트 고객센터](https://cashnote.zendesk.com/hc/ko/articles/360003495872)

---

### 3.2 머니핀 (모두의회계)

**개요**: 딥러닝 기반 태그형 셀프 회계 서비스

**장점**:
- 부가세 신고 시만 **3만원** (평소 무료)
- AI 회계엔진 '핀키'로 자동 복식부기
- 5인 미만 초기 사업자 최적화
- 인건비 신고까지 월 1.5만원

**단점**:
- API 미제공
- 직접 입력 필요한 부분 있음
- 세무사 연동 시 추가 비용

**적합도**: ⭐⭐⭐⭐⭐ (셀프 기장 + 세금신고)

> 출처: [THE VC - 모두의회계](https://thevc.kr/moduaccounting), [중소기업 세무회계 솔루션 비교](https://www.jounboo.com/blog/top-10-accounting-solutions-for-small-businesses-pros-and-cons-comparison-2024)

## 6. 볼타 API 연동 예시 코드

```javascript
// services/boltaService.js
const axios = require('axios');

const BOLTA_API_URL = 'https://api.bolta.io/v1';
const BOLTA_API_KEY = process.env.BOLTA_API_KEY;

/**
 * 세금계산서 발행
 */
async function issueTaxInvoice(invoiceData) {
  try {
    const response = await axios.post(
      `${BOLTA_API_URL}/tax-invoices`,
      {
        supplierBusinessNumber: process.env.BUSINESS_NUMBER,
        buyerBusinessNumber: invoiceData.buyerBizNo,
        buyerName: invoiceData.buyerName,
        items: [{
          name: invoiceData.itemName,
          quantity: 1,
          unitPrice: invoiceData.amount,
          taxAmount: Math.floor(invoiceData.amount * 0.1)
        }],
        issueDate: new Date().toISOString().split('T')[0]
      },
      {
        headers: {
          'Authorization': `Bearer ${BOLTA_API_KEY}`,
          'Content-Type': 'application/json'
        }
      }
    );

    console.log('[Bolta] 세금계산서 발행 성공:', response.data.invoiceId);
    return response.data;

  } catch (error) {
    console.error('[Bolta] 세금계산서 발행 실패:', error.message);
    throw error;
  }
}

module.exports = { issueTaxInvoice };
```
