## 토론 시스템

### 시작 방법
```
/토론 [주제]
```

### 토론 진행
1. 각 팀원이 역할에 맞는 의견 제시
2. CEO/CRO 피드백
3. 결론 도출

### 결정 기록
```
/결론
```

### 결정문 형식
```
DEC-YYYY-MMDD-NNN_[제목].md
```
