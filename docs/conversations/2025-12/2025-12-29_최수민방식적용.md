## 2. 핵심 인사이트

### 컨텍스트 관리 3원칙
| 원칙 | 설명 |
|------|------|
| 스크립트 추출 | 데이터는 스크립트로 먼저 추출 |
| 배치 분할 | 대량 작업은 10개씩 분할 |
| 파일 저장 | 중간 결과물은 파일로 저장 |

### 피드백 루프
- 별도 문서 X → 에이전트 .md에 직접 반영
- AI가 다음부터 자동 학습
- 한 번 수정으로 끝내지 않고 지침에 영구 반영

### 자가 검증 4단계
1. **도그푸딩** - 만든 사람이 첫 번째 사용자
2. **검증 로직** - 스킬에 검증도 포함
3. **지침 보강** - 오류 발견 → 즉시 반영
4. **최종 종합** - 파이프라인 끝에서 전체 검증

### 지식 관리
- 저장만으론 부족, **활용**이 핵심
- 검색 + 분석 + 컨텍스트 빌드
