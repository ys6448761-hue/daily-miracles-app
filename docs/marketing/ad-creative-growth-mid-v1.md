## 1. PRODUCTION SUMMARY

| 항목 | 내용 |
|------|------|
| **Title** | Growth-Mid — "멈춰 있는 기분, 익숙하죠." |
| **Objective** | Growth-Builder × Mid LTV 정체 공감 → 쉬운 첫 행동(3분 진단) 전환 |

| **Tone** | 방향 제시 + 쉬운 첫 행동 (진입장벽 낮춤) |

| **Growth-Mid** | **growth_builder** | **mid** | **방향 제시 + 낮은 진입장벽** | **정체 → "3분이면 돼" 전환** |

| 항목 | Growth-High | Growth-Mid |

| 감정 강도 | 확신 (determined) | 안심 (reassured) |
