## 6. 핵심 서비스 기능

### 6-1. 기적지수 (Miracle Score) — 50~100점

소원이가 작성한 소원을 AI가 분석하여 **5가지 차원**의 잠재력 점수를 산출합니다:
- **현재 상황 인식** — 자신의 위치를 얼마나 잘 파악하고 있는지
- **개선 의지** — 변화를 향한 내적 동기 수준
- **환경적 지지** — 주변 환경이 얼마나 지지적인지
- **실행 가능성** — 구체적 행동으로 옮길 수 있는 정도
- **내용 구체성** — 소원이 얼마나 명확하게 표현되었는지

점수와 함께 **에너지 유형(보석)**이 부여됩니다:

| 보석 | 의미 | 상징 |
|------|------|------|
| ❤️ 루비 | 열정 | 강렬한 실행력 |
| 💙 사파이어 | 안정 | 차분한 지속력 |
| 💚 에메랄드 | 성장 | 자연스러운 발전 |
| 💎 다이아몬드 | 명확 | 투명한 목표 의식 |
| 💛 시트린 | 긍정 | 밝은 에너지 |

### 6-2. 신호등 시스템 (안전 장치)

모든 소원은 접수 즉시 자동 분류됩니다:
- 🟢 **GREEN** — 일반 소원 → 즉시 자동 처리
- 🟡 **YELLOW** — 민감 주제 (건강/재정/법률) → 24시간 내 전문 팀 검토
- 🔴 **RED** — 위기 감지 (자해/자살 키워드) → 즉시 전문 상담 연결, 자동 발송 차단

→ 마케팅 메시지에서는 "안전하고 전문적인 관리 시스템" 정도로 언급

### 6-3. 소원그림 패키지

소원이의 여정을 기념하는 시각 콘텐츠 2종:
- **탑승증명서 (Certificate)** — 4:5 비율, 그라데이션 배경 + 탑승일/소원ID + AI 캡션 (22자 이하)
- **엽서 (Postcard)** — 1:1 정방형, 풀블리드 이미지 + 미니멀 씰 + AI 캡션, SNS 공유 최적화

→ 각자의 에너지 톤에 맞춘 자동 캡션으로 세상에 하나뿐인 기념품

### 6-4. 워밍업(WU) 자기탐색

30분 가이드형 질문 여정을 통한 자기 이해 프로그램:
- **관계 탐색 (REL)** — 관계 패턴과 소통 스타일 이해
- **자아 탐색 (SELF_ST_TXT)** — 내면의 강점과 성장 방향 발견

완료 시 AI가 따뜻한 격려 + 심리학적 인사이트 + 5대 에너지 피트니스 점수 제공

### 6-5. 7일 응원 메시지

소원 접수 후 7일간 매일 아침/저녁 맞춤 응원 메시지 발송:
- 카카오 알림톡 또는 SMS
- 소원이의 에너지 유형에 맞춘 톤
- 매일 다른 실천 포인트 제시

### 6-6. 30일 맞춤 로드맵 (PDF)

기적지수 분석 결과를 바탕으로 생성되는 개인 맞춤 실천 계획서

## 7. 소원이 여정 (User Journey)

```
① 소원 접수 → ② 안전 분류(신호등) → ③ 기적지수 분석 → ④ 소원그림 생성
→ ⑤ 접수 확인 메시지 발송 (카카오/SMS) → ⑥ 7일간 응원 메시지
```

마케팅에서 강조할 포인트:
- **즉시성**: 소원 제출 즉시 분석 시작
- **안전성**: RED 신호 시 전문가 즉시 개입
- **개인화**: 보석 유형 + 기적지수 기반 맞춤 콘텐츠
- **지속성**: 7일간 매일 응원, 30일 로드맵
