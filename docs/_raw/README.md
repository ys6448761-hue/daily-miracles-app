# docs/_raw/ — CEO 빠른 적재 폴더

> 여기에 던지면 코미가 알아서 분류하고 승격합니다.
> CEO는 이 폴더에 파일을 넣기만 하면 됩니다.

## 파일명 규칙

```
YYYY-MM-DD_제목.md
```

예: `2026-02-21_섬박람회-현장운영-메모.md`

## 작성 형식

자유. 아래 중 아무거나:
- 메모 형태
- 회의록
- 아이디어 노트
- 복붙한 카톡/메일 내용
- AI 대화 결과 복사

## 투입 후 흐름

```
CEO가 여기에 파일 넣음
     ↓
코미가 다음 세션에서 감지
     ↓
승격 레벨 판단 (L1/L2/L3)
     ↓
해당 위치로 이동 + MASTER-CONTROL 업데이트
     ↓
_raw 원본에 promoted 마커 추가
```

## promoted 마커

승격 완료된 파일 상단에 아래가 추가됩니다:

```
<!-- PROMOTED: L2 → docs/decisions/DEC-2026-0221-001.md -->
<!-- PROMOTED_AT: 2026-02-21 -->
```

이 마커가 있는 파일은 재승격하지 않습니다.
