## EP02 제작자를 위한 인수인계

EP02를 시작하기 전 아래 순서로 읽는다.

1. `SSOT-CHAR-001_Sowoni_Character_Bible.md` — 소원이 캐릭터 기준 확인
2. `SSOT-OPS-001_DreamTown_Video_Pipeline.md` — 파이프라인 전 과정 확인
3. `STATE-EP01_Where_My_Heart_Became_A_Star.md` — EP01 감정 상태 이어받기

> 소원이는 EP01에서 **여수의 바람**을 획득했다.
> EP02의 목표는 **마음을 열기**다.
> 감정 진행도는 희망이 슬픔을 넘어서는 방향으로 설계한다.
