# AIL-STORY-2026-0404-001 — 나의 이야기 초안 MVP 검증

**유형**: AIL (Assumption, Insight, Learning) 검증 실험
**상태**: 진행 중
**생성일**: 2026-04-04
**담당**: Claude Code
**검증 대상 경로**: `/story-draft-mvp`

---

## 목적

"나의 이야기 초안" 화면이 감정 반응을 일으키는지 검증한다.
이번 MVP는 실제 판매 퍼널이 아니다.
사용자가 **화면에서 멈추는지**, **자기 이야기로 인식하는지** 확인하는 것이 목표다.

---

## 검증 가설 (Assumption)

| # | 가설 | 검증 방법 |
|---|------|-----------|
| A1 | 스크롤형 이야기 화면을 보면 감정이 올라온다 | 진행자 관찰 + 반응 수집 |
| A2 | "이거 내 이야기 같다 / 책 같다" 반응이 나온다 | 자유 발화 + 인터뷰 |
| A3 | 다시 보고 싶어지는 마음이 생긴다 | "이어가기" 클릭률 관찰 |

---

## 테스트 질문 (운영자용)

1. 이거 보고 어떤 느낌이 들었나요?
2. 이걸 다시 보고 싶은 마음이 드나요?
3. 이 이야기를 이어가보고 싶은 생각이 드나요?

---

## 범위

### 포함 (In-Scope)
- 정적 HTML/JSX MVP 화면 (샘플 데이터 하드코딩)
- 스크롤형 4단 구조: 인트로 → 시작 → 변화 → 전환 → 다음
- 선택지 2개 (동일 위계): "간직하기" / "이어가기"
- 최소 로그 3개: 진입 / 간직하기 클릭 / 이어가기 클릭

### 제외 (Out-of-Scope)
- 실제 상담 플로우 연결
- 결제 / 가격 문구
- 자동 생성 파이프라인 / LLM 연동
- 실제 사용자 데이터 불러오기
- 관리자 기능
- 출판 / 상품 설명
- 여수 연결
- 6개월 / 1년 전환 퍼널 본 구현
- DB 마이그레이션
