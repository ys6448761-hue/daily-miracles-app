## 🎯 새로운 소통 방식

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  【기존 방식】                                                              │
│  코미가 질문서 작성 → 팀원들 답변 → 코미가 종합                            │
│  → 코미의 "해석"이 들어갈 수 있음                                          │
│                                                                             │
│  【새로운 방식】 ⭐                                                         │
│  푸르미르님이 직접 각 팀원에게 "/토론 [주제]" 전달                         │
│  → 각자 토론 & 결론 도출                                                   │
│  → 푸르미르님이 결론을 코미에게 전달 (택배!)                               │
│  → 코미가 종합 분석 & 최종 결정안                                          │
│                                                                             │
│  ─────────────────────────────────────────────────────────────────────────  │
│                                                                             │
│  💡 핵심 변화:                                                             │
│  • 각 팀원이 "직접" 자신의 관점으로 토론                                   │
│  • 코미의 해석/추측 없이 순수한 의견                                       │
│  • 더 풍부하고 다양한 아이디어!                                            │
│  • 진짜 "살아있는 팀" 느낌!                                                │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 📝 /토론 사용법

### 1️⃣ 푸르미르님이 주제를 전달합니다

```
푸르미르: /토론 [주제]

예시:
• /토론 프리미엄 가격을 얼마로 할까?
• /토론 베타 테스터 첫 타겟은 누구로?
• /토론 런칭 날짜를 앞당길까?
```

### 2️⃣ 여러분은 자신의 관점에서 토론합니다

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  🎨 재미라면:                                                              │
│  → 크리에이티브, 감성, UX, 바이럴, 브랜딩 관점                             │
│                                                                             │
│  📊 루미라면:                                                              │
│  → 데이터, 효율, KPI, 비용, 기술적 가능성 관점                             │
│                                                                             │
│  💎 여의보주라면:                                                          │
│  → 철학, 진정성, 소원이 경험, 의미 관점                                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 3️⃣ 결론을 명확하게 정리합니다

```
【결론 형식】

【주제】: [토론 주제]

【나의 입장】: 찬성 / 반대 / 조건부 찬성 / 기타

【핵심 의견】:
• (2-3줄로 핵심만!)

【제안/대안】:
• (있다면)

【우려 사항】:
• (있다면)

【한 줄 요약】:
"                                            "
```

### 4️⃣ 푸르미르님이 코미에게 전달합니다

```
푸르미르님이 "택배" 역할!
→ 각 팀원의 결론을 복사해서 코미에게 전달
→ 코미가 종합 분석 & 최종 결정안 도출
→ 푸르미르님 최종 승인!
```
