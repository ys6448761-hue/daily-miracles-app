## 💡 핵심 인사이트

### GitHub vs Code 저장 차이
- **저장 위치**: 둘 다 동일 (GitHub)
- **차이점**: 누가 저장하느냐만 다름
- **결론**: 병행 사용 (간단한 건 직접, 정리 필요하면 Code)

### GitHub 용량/비용
- 현재 사용: ~0.5MB (한도의 0.05%)
- 10년 예상: ~150MB (한도의 15%)
- **결론**: 마크다운만 저장하면 영원히 무료!

### 푸르미르님 새 워크플로우
```
아침: Code에게 "브리핑 줘" (1분)

새 세션: 브리핑 복사 → 팀원에게 전달
```
