## 📋 **코드 우선 활용 기준 (5가지)**

### 1️⃣ 전체 맥락 필요

```
상황:
"우리가 어떤 방향으로 가고 있지?"
"프로젝트 전체 그림이 궁금해"

→ 코드 먼저!

이유:
코드는 모든 대화를 기억
전체 흐름을 가장 잘 파악
```

### 2️⃣ 과거 논의 참고

```
상황:
"이전에 이거 논의했나?"
"예전에 결정한 게 뭐였지?"

→ 코드 먼저!

이유:
코드는 히스토리 완벽 보유
누락 없이 모두 검색 가능
```

### 3️⃣ 기술 구현 필요

```
상황:
"이거 어떻게 만들지?"
"MVP에 이 기능 추가하려면?"

→ 코드 먼저!

이유:
코드가 실제 구현 담당
기술적 제약 가장 잘 앎
```

### 4️⃣ 복잡한 분석 필요

```
상황:
"사용자 심리 분석해줘"
"경쟁사 비교 분석 필요해"

→ 코드 먼저!

이유:
코드가 여러 자료 종합 가능
깊이 있는 분석 가능
```

### 5️⃣ 의사결정 전 검토

```
상황:
"이 방향 맞나?"
"이거 괜찮은 아이디어야?"

→ 코드 먼저!

이유:
코드가 전체 프로젝트 맥락
과거 결정들과 정합성 체크
```
