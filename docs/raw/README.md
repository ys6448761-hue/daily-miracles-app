# Raw Conversations (원본 대화)

> Claude Code 대화 원본을 그대로 보관하는 폴더입니다.

## 폴더 구조

```
docs/raw/
├── README.md (이 파일)
└── conversations/
    └── 2025-12/
        └── 2025-12-29_full.md  ← 전체 대화 원본
```

## 저장 방법

### 방법 1: VS Code에서 직접 (추천!)

1. 클로드 대화 전체 선택 (Ctrl+A)
2. 복사 (Ctrl+C)
3. VS Code에서 새 파일 생성
4. 붙여넣기 (Ctrl+V) + 저장 (Ctrl+S)
5. Git 커밋 & 푸시

### 방법 2: GitHub 웹에서 업로드

1. GitHub 저장소 열기
2. docs/raw/ 폴더로 이동
3. "Add file" → "Upload files"
4. 파일 드래그 & 드롭
5. Commit

## 파일명 규칙

```
YYYY-MM-DD_주제.md

예시:
- 2025-12-29_full.md (하루 전체)
- 2025-12-29_최수민방식.md (주제별)
```

## 원본 vs 요약

| 구분 | 위치 | 용도 |
|------|------|------|
| 원본 | `docs/raw/conversations/` | 전체 내용 보관, 검색 |
| 요약 | `docs/conversations/` | 핵심만 정리, 빠른 참조 |

## 왜 원본을 보관하나?

1. **컨텍스트 복원** - 요약에서 놓친 맥락 확인
2. **검색** - 키워드로 과거 대화 찾기
3. **학습** - 어떤 질문이 좋은 답변을 얻었는지
4. **백업** - 중요 대화 영구 보관
