## 1. 배경 및 목적

소원드림광장(소원꿈터 광장)이 개발 완료되었습니다.
이를 바탕으로 **dailymiracles.kr(Wix) 홈페이지를 소원드림광장 중심으로 재구성**하기 위해

서로 공감하며 함께 응원하는 **커뮤니티 피드 공간**입니다.

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 소원꿈터 (sowon-dreamtown) |
| 기술 스택 | Next.js 16 + Prisma 6 + Tailwind CSS |
| 인증 방식 | 닉네임 기반 간편 로그인 |
| 현재 상태 | MVP v0 개발 완료, Render 배포 예정 |
| 스타일 | 기본 흰 배경 + Tailwind (디자인 추가 작업 가능) |

## 7. 루미 홈페이지 재구성 시 제안 포인트

### 핵심 방향
> **소원드림광장 = 서비스의 메인 진입점**
> 방문자가 홈페이지에 들어왔을 때 가장 먼저 광장의 실제 소원글 피드를 보여주는 구조

### 고려할 요소

1. **광장 미리보기 섹션**
   - 최근 승인된 소원글 3~5개를 홈페이지에 임베드 또는 링크
   - "나도 소원 올리기" CTA 버튼으로 회원가입 유도

2. **이벤트/미션 섹션**
   - 현재 진행 중인 이벤트를 배너 형태로 노출
   - 참여 유도 → 앱으로 유입

3. **서비스 소개 재배치**
   - 기존 설명 중심 → 실제 광장 콘텐츠 + 간단한 서비스 소개로 전환

4. **CTA 흐름 설계**
   - 방문 → 소원글 구경 → 로그인 → 내 소원 올리기 → 이벤트 참여

5. **현재 제약 사항**
   - 이미지 없음 (텍스트 카드 형태)
   - Wix ↔ 앱 간 실시간 피드 연동은 별도 기술 검토 필요
   - 앱 URL: 배포 후 확정 (현재 로컬 개발 중)

### Q1. 소원꿈터 → Render 배포 일정

**현재 준비 상태**: 기능 완성, 배포 설정 미완성

완료된 것:
- 앱 코드 100% 완성 (MVP v0)
- `next build` / `next start` 스크립트 준비됨
- 로컬 SQLite 동작 확인됨

배포 전 필수 작업:

| 작업 | 예상 시간 |
|------|----------|
| Render PostgreSQL DB 생성 | 10분 |
| 환경변수 설정 (DATABASE_URL, NEXTAUTH_SECRET 등) | 20분 |
| `schema.prisma` provider를 `postgresql`로 전환 | 10분 |
| Prisma 마이그레이션 운영 DB에 적용 | 10분 |
| Render Web Service 생성 + 빌드 확인 | 30분~1시간 |

**결론**: 준비되면 당일 배포 가능. 총 1~2시간 작업.
빌드 오류/DB 연결 이슈 발생 가능성 고려해 여유있게 **반나절** 확보 권장.

### Q2. 앱 통합형(옵션 B) 기술 난이도

> 기존 `app.dailymiracles.kr`(Express)에 랜딩 페이지를 합치는 게 현실적인지

**결론: 비현실적. 권장하지 않음.**

이유:
- `app.dailymiracles.kr` = **Express.js** (Node.js, 서버사이드 렌더링 없음)
- 소원꿈터 = **Next.js** (완전히 다른 프레임워크)
- 두 앱을 하나로 합치려면 사실상 **전체 재작성** 수준의 작업
- Render에서 두 서비스를 같은 도메인으로 묶는 것도 복잡한 리버스 프록시 설정 필요

**현실적인 대안 — 서브도메인 분리**:
```
app.dailymiracles.kr    → 기존 Express 앱 (소원 신청 등 기존 기능 유지)
plaza.dailymiracles.kr  → 소원꿈터 (Next.js, 신규 서비스)
```

### Q3. Wix → 앱 리다이렉트 구조

**Wix 처리 방법 3가지 옵션**:

| 옵션 | 방법 | 난이도 | 추천 |
|------|------|:------:|:----:|
| A. Wix 유지 + 링크만 추가 | Wix 버튼에 `plaza.dailymiracles.kr` URL 연결 | ★☆☆ | ✅ 최우선 |
| B. Wix에서 iframe 임베드 | Wix 페이지에 소원꿈터 iframe 삽입 | ★★☆ | 조건부 |
| C. Wix 폐기 + 도메인 리다이렉트 | `dailymiracles.kr` → `plaza.dailymiracles.kr` | ★☆☆ | 나중에 |

**옵션 A 상세 (권장)**:
```
dailymiracles.kr (Wix)
  └─ 홈페이지에 "소원드림광장 바로가기" 버튼 추가
       → plaza.dailymiracles.kr 이동
```
Wix는 그대로 두고, 소원꿈터 배포 후 링크만 추가하면 됩니다.
Wix 폐기 여부는 소원꿈터 안정화 이후 별도로 결정해도 됩니다.
