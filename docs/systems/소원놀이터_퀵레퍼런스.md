## 피드 랭킹 공식

```
랭킹 = 0.55 x 철학점수 + 0.35 x 도움점수 + 0.10 x 신선도
```

---

## 감점 키워드

**압박 표현** (-5점씩)
- 반드시 해야, 무조건 성공, 실패하면 안

**낙인 표현** (-5점씩)
- 게으른, 무능한, 루저, 패배자

**하드블록** (즉시 차단)
- 자해/자살, 폭력/협박, 혐오표현, 불법

## 필수 필드 (창작물)

```

one_step       → 오늘 할 일
```

## 주요 API

| 기능 | 엔드포인트 |
|------|-----------|
| 창작물 생성 | `POST /api/playground/artifacts` |
| 메인 피드 | `GET /api/playground/feed` |
| 하이라이트 | `GET /api/playground/feed/highlights` |
| 반응 추가 | `POST /api/playground/artifacts/:id/reactions` |
| 공유 생성 | `POST /api/playground/shares` |
| 보상 현황 | `GET /api/playground/users/:id/rewards` |
