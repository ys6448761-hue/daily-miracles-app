## 5. Wix 디자인 가이드

### 컬러 팔레트

| 용도 | 컬러명 | HEX |
|------|--------|-----|
| 배경 (깊은 바다) | 딥 인디고 | `#1A237E` |
| 강조 (산호빛) | 코랄 핑크 | `#FF7F7F` |
| 포인트 (별빛) | 스타 골드 | `#FFD54F` |
| 서브 (용궁 기둥) | 터콰이즈 | `#4DD0E1` |
| 신비로움 | 소프트 퍼플 | `#B39DDB` |
| 텍스트 (밝음) | 웜 화이트 | `#FFF8E1` |

### 폰트 권장

| 용도 | 폰트 |
|------|------|
| 제목 | Noto Sans KR Bold |
| 본문 | Noto Sans KR Regular |
| 강조 | Noto Sans KR Medium |

### 이미지 배치

```
[Hero Section]
├── 배경: 용궁 일러스트 또는 그라데이션 (딥 인디고 → 터콰이즈)
└── 단체샷 이미지: aurora5_team_dragon_1024.png

[팀원 카드]
├── 각 카드에 개별 캐릭터 이미지
├── 이미지 크기: 256x256 또는 512x512
└── 배경: 각 캐릭터 메인 컬러 그라데이션

[철학 섹션]
└── 배경: 별빛 + 바다 패턴 또는 단색
```
