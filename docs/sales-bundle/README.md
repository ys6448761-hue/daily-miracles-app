## 빠른 참조

### 영업 프로세스

```
1. 초기 접촉 → 카드 1 (문제 공감)
2. 관심 표현 → 카드 2 (효율 강조) + 데모 제안
3. 데모 진행 → 02_Pilot (8분 스크립트)
4. 결정 단계 → 카드 3 (신뢰 구축) + 01_Approval
5. 파일럿 시작 → 자료요청서 발송
```

### 공유용 URL

```
https://app.dailymiracles.kr/yeosu-ops-center/
```

### 핵심 메시지 3개

1. **SSOT** → "최신본이 뭐예요?" 질문 제로
2. **1클릭** → 결과보고 3일 → 1분
3. **무상 파일럿** → 리스크 없이 검증
