# 결재 요청서

## 여수시 MICE 인센티브 운영 컨트롤타워 도입 건

---

### 1. 요약

| 항목 | 내용 |
|------|------|
| **건명** | MICE 인센티브 운영 컨트롤타워 시스템 도입 |
| **목적** | 행사 운영 혼선 최소화 + 결과보고 자동화 |
| **대상** | 여수시 관광과 MICE 인센티브 지원사업 |
| **기간** | 파일럿 3개월 → 본도입 판단 |
| **비용** | 파일럿 무상 / 본도입 시 협의 |

---

### 2. 현황 및 문제점

| 문제 | 영향 | 빈도 |
|------|------|------|
| "최신본이 뭐예요?" | 운영 혼선, 민원 | 매 행사 |
| 변경 이력 분실 | 책임 소재 불명확 | 빈번 |
| 결과보고 수작업 | 담당자 야근, 오류 | 매 정산 |
| 증빙 누락 | 정산 지연/반려 | 30%+ |

---

### 3. 솔루션 핵심 기능

| 기능 | 효과 |
|------|------|
| **SSOT 보드** | 승인된 최신본만 공식 → 혼선 제로 |
| **변경이력 100%** | 누가/언제/무엇을 → 분쟁 예방 |
| **자동 체크리스트** | 필수 증빙 누락 방지 |
| **1클릭 패키지** | 결과보고 ZIP 자동 생성 |

---

### 4. 기대 효과

| 지표 | 현재 | 목표 | 측정 방법 |
|------|------|------|----------|
| 운영 문의 | 10건+/행사 | 3건 이하 | 문의 건수 |
| 결과보고 작성 | 3일+ | 당일 | 작성 시간 |
| 증빙 누락률 | 30%+ | 5% 이하 | 반려 건수 |
| 승인 리드타임 | 측정불가 | 실시간 KPI | 시스템 |

---
