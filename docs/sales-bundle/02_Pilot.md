# 파일럿 제안서

## 여수시 MICE 운영 컨트롤타워 파일럿 프로그램

---

### 1. 제안 요약

| 항목 | 내용 |
|------|------|
| **프로그램명** | MICE 운영 컨트롤타워 파일럿 |
| **기간** | 3개월 (1개 행사 사이클) |
| **대상 행사** | MICE 인센티브 지원 행사 1건 |
| **비용** | **무상** (검증 목적) |
| **목표** | 운영 혼선 50% 감소 + 결과보고 당일 생성 |

---

### 2. 파일럿 범위

**포함**
- ✅ 시스템 계정 발급 (5명)
- ✅ 1시간 온보딩 교육
- ✅ SSOT 보드 + 승인 워크플로우
- ✅ 결과보고 패키지 자동 생성
- ✅ 상시 기술 지원

**미포함**
- ❌ 커스터마이징 개발
- ❌ 타 시스템 연동
- ❌ 오프라인 교육 (온라인만)

---

### 3. 성공 기준

| 지표 | 기준 | 측정 방법 |
|------|------|----------|
| 시스템 활용률 | 주 3회 이상 접속 | 로그 분석 |
| 변경 추적률 | 100% 기록 | 감사로그 |
| 체크리스트 충족 | 5/5 필수항목 | 시스템 |
| 사용자 만족도 | 4점/5점 이상 | 설문 |

---

### 4. 일정

| 주차 | 활동 | 산출물 |
|------|------|--------|
| 1주 | 킥오프 + 온보딩 | 계정 발급, 교육 완료 |
| 2주 | 행사 정보 입력 | SSOT 초기 설정 |
| 3~6주 | 행사 진행 중 운영 | 변경이력 누적 |
| 7~8주 | 행사 종료 + 결과보고 | ZIP 패키지 생성 |
| 9~12주 | 피드백 + 개선 | 성과 보고서 |

---

### 5. 필요 자원

| 자원 | 수량 | 비고 |
|------|------|------|

# 8분 데모 스크립트

## 데모 목표
> "SSOT로 혼선 제로, 1클릭으로 결과보고 완성"

---

## 타임라인 (총 8분)

### 0:00-0:30 | 오프닝 (30초)

**멘트:**
> "안녕하세요. 오늘 여수시 MICE 운영에 도움이 될 시스템을 보여드리겠습니다.
> 핵심은 두 가지입니다. 첫째, '최신본이 뭐예요?'라는 질문이 사라집니다.
> 둘째, 결과보고서가 버튼 하나로 만들어집니다."

**화면:** 메인 화면 (행사 목록)

---

### 0:30-2:00 | SSOT 보드 (1분 30초)

**멘트:**
> "먼저 SSOT 보드입니다. Single Source of Truth, 유일한 진실의 원천이라는 뜻입니다.
> 여기 보이는 항목들이 공식 운영 정보입니다. 승인된 내용만 표시됩니다."

**액션:**
1. 행사 클릭 → 상세 화면
2. SSOT 보드 탭 클릭
3. 카테고리별 항목 확인 (일정, 운영, 공지)

**멘트:**
> "예를 들어 '개막식 시간'을 변경해 보겠습니다.
> 19시에서 19시 30분으로 바꾸고... 사유를 입력하면...
> 자동으로 승인 요청이 생성됩니다."

**액션:**
4. 항목 클릭 → 수정 → 저장
5. 승인 대기 상태 확인

---

### 2:00-3:30 | 승인 워크플로우 (1분 30초)

**멘트:**
> "이제 승인함을 보겠습니다.
> 방금 변경한 내용이 대기 중으로 들어와 있습니다.
> 담당자가 확인하고 승인하면..."

**액션:**
1. 승인함 탭 클릭
2. 대기 중인 요청 확인
3. 승인 버튼 클릭
4. 사유 입력 → 승인

**멘트:**
> "이렇게 승인되면 공식 값으로 반영됩니다.
> 누가, 언제, 왜 변경했는지 전부 기록됩니다.
> 나중에 '누가 바꿨어요?'라는 질문에 바로 답할 수 있습니다."

---

### 3:30-5:00 | 감사 로그 (1분 30초)

**멘트:**
> "그 기록이 바로 여기, 감사 로그입니다.
> 모든 변경 이력이 시간순으로 쌓입니다."

**액션:**
1. 감사 로그 탭 클릭
2. 필터 적용 (오늘, UPDATE)
3. 상세 내용 확인

**멘트:**
> "필터로 원하는 기간, 액션 유형을 볼 수 있고요.
> CSV로 내보내기도 됩니다. 보고서 작성할 때 유용합니다."

**액션:**
4. CSV 내보내기 클릭

---

### 5:00-6:30 | 결과보고 체크리스트 (1분 30초)

**멘트:**
> "이제 MICE 인센티브의 핵심, 결과보고입니다.
> 체크리스트를 보시면 필수 서류 5개가 있습니다."

**액션:**
1. KPI 리포트 또는 MICE 탭 클릭
2. 체크리스트 확인

**멘트:**
> "참가자 등록부, 숙박확인서, 지출증빙, 사진 2종...
> 지금 보시면 3개가 완료되어 있고 2개가 부족합니다.
> 시스템이 자동으로 체크해주니까 누락될 일이 없습니다."

---

### 6:30-7:30 | 패키지 생성 (1분)

**멘트:**
> "필수 항목이 다 채워지면 버튼 하나로 패키지가 만들어집니다.
> 생성 버튼을 누르면..."

**액션:**
1. 패키지 생성 버튼 클릭 (시연용)
2. 결과 확인

**멘트:**
> "이렇게 ZIP 파일이 만들어집니다.
> 안에는 결과보고서 HTML, 참가자 CSV, 숙박 CSV, 지출 CSV,
> 사진대장, 체크리스트가 전부 들어 있습니다.
> 3일 걸리던 작업이 1분이면 끝납니다."

---

### 7:30-8:00 | 클로징 (30초)

**멘트:**
> "정리하면, SSOT로 혼선 제로, 감사로그로 책임 명확,
> 체크리스트로 누락 방지, 1클릭으로 결과보고 완성.
>
> 파일럿은 무상으로 진행됩니다.
> 3개월 동안 행사 하나에 적용해보시고 판단하시면 됩니다.
>
> 질문 있으시면 말씀해 주세요."

**화면:** 메인 화면 또는 연락처
