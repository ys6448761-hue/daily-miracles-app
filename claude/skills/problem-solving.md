## 개요

**문제해결 소원**은 현재 겪고 있는 문제나 고민을 AI가 분석하여
구체적인 해결책과 다음 행동을 제시하는 서비스입니다.

과거/현재 문제에 초점을 맞추며, 즉시~1개월 내 실행 가능한 솔루션을 제공합니다.

## 대상

- 직장/업무 고민이 있는 직장인
- 인간관계 문제를 겪는 분
- 진로/커리어 고민이 있는 분
- 일상적인 결정에 도움이 필요한 분

## 흐름

1. **닉네임 입력** - 익명으로 진행 가능
2. **고민 요약** - 핵심 문제 한 줄 요약
3. **상황 설명** - 구체적인 상황 서술
4. **시도 경험** - 이미 시도해본 것들
5. **제약사항** - 피하고 싶은 것, 지키고 싶은 것
6. **초점 선택** - 원하는 분석 방향
7. **AI 분석** - GPT-4 기반 분석 (25~30초)
8. **결과 제공** - 인사이트, 선택지, 다음 행동 제시

## 입력

```json
{
  "nickname": "달빛고래",
  "wishSummary": "상사가 무시해요...",
  "situation": "스타트업 PM으로...",
  "tries": "개인 면담 시도...",
  "constraints": "퇴사는 피하고...",
  "focus": "당장 뭘 할지..."
}
```

## 출력

```json
{
  "category": "직장/업무",
  "coreIssue": "조직 내 영향력 부족",
  "emotionalPattern": "답답함, 무력감",
  "insights": ["인사이트 1", "인사이트 2", "인사이트 3"],
  "options": ["선택지 1", "선택지 2"],
  "nextActions": ["다음 행동 1", "다음 행동 2", "다음 행동 3"],
  "warnings": ["주의사항"]
}
```

| 티어 | 가격 | 포함 |

| 무료 | $0 | 1회 분석, 기본 리포트 |

## 기술 스택

- **AI 모델**: GPT-4
- **응답 시간**: 25~30초
- **카테고리 자동 감지**: 직장, 인간관계, 진로, 건강, 재정 등

## 연동 서비스

- → `wish-voyage.md` (소원실현 항해와 선택 가능)
- → `message-dispatcher` (후속 알림톡 발송)
