## 개요

**온라인 소원이어가기**는 오프라인 소원교환체험에서 받은 QR 코드를 스캔하여
온라인으로 소원을 이어가는 서비스입니다.

두 가지 경로 중 선택할 수 있습니다:
1. **문제해결 소원** - 현재 문제 중심
2. **소원실현 항해** - 미래 목표 중심

## 대상

- 오프라인 소원교환체험 참가자
- QR 코드 카드를 받은 분
- 온라인으로 심화 분석을 원하는 분

## 흐름

1. **QR 스캔** - 오프라인에서 받은 QR 코드 스캔
2. **랜딩 페이지** - `daily-miracles.com/wish?id=ABC123`
3. **경로 선택** - 문제해결 or 소원실현 중 선택
4. **폼 작성** - 선택한 경로에 맞는 상세 입력
5. **AI 분석** - GPT-4 기반 분석 (25~35초)
6. **결과 확인** - 리포트 페이지에서 결과 확인
7. **후속 서비스** - 7일 응원 메시지, 체크인 알림

## 입력

**경로 선택 전:**
- QR 코드 ID (URL 파라미터)

**문제해결 선택 시:**
→ `problem-solving.md` 입력 형식

**소원실현 선택 시:**
→ `wish-realization.md` 입력 형식

## 출력

**문제해결 선택 시:**
- 분석 리포트 (카테고리, 핵심이슈, 인사이트, 선택지, 다음행동)
- 리포트 ID: `report_xxxxx`

**소원실현 선택 시:**
- 항해 계획 (항해지수, 액션플랜, SWOT, 체크포인트)
- 리포트 ID: `voyage_xxxxx`

## 가격

- **무료**: 1회 분석
- **프리미엄**: 무제한 분석 + 상세 리포트

## 연동 서비스

- ← `wish-exchange.md` (오프라인 소원교환)
- → `problem-solving.md` (문제해결 경로)
- → `wish-realization.md` (소원실현 경로)
- → `miracle-analyzer` (기적지수 분석)
- → `message-dispatcher` (알림 발송)

## 기술 연동

| 단계 | API | 설명 |
|------|-----|------|
| 경로1 | POST `/api/problem/online-wish` | 문제해결 분석 |
| 경로2 | POST `/api/wish-voyage/online-wish` | 소원실현 분석 |
| 결과1 | GET `/problem-result?reportId=xxx` | 문제해결 결과 |
| 결과2 | GET `/voyage-result?reportId=xxx` | 소원실현 결과 |
